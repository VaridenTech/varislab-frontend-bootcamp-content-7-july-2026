สรุปบทเรียน TypeScript และก้าวต่อไป

ยินดีด้วย! เราเดินทางจากการทำความรู้จัก TypeScript ครั้งแรก มาจนถึงการนำไปใช้งานร่วมกับ Ecosystem จริงแล้ว มาทบทวนภาพรวมทั้งหมดที่ผ่านมากันอีกครั้ง

JavaScript เทียบกับ TypeScript

  • Type แบบ Dynamic และ Weak ใน JavaScript คือที่มาของ Bug ที่มักโผล่มาให้เห็นตอนรันจริงเท่านั้น
  • TypeScript คือ Superset ของ JavaScript ที่เพิ่มการตรวจสอบ Type ตั้งแต่ก่อนรันโค้ด
  • ติดตั้งและใช้งานผ่านคำสั่ง tsc — ตรวจสอบได้แม้แต่ไฟล์ .js เดิมด้วย --checkJs

ไวยากรณ์หลักของ TypeScript

  • Type Annotation สำหรับตัวแปร Function Parameter และค่าที่ Return
  • Primitive Types, Literal Types, Array Types, Object Types
  • Union Types และการ Narrow ขอบเขต Type ด้วย if หรือ typeof
  • การสร้าง Type ของตัวเองด้วย type Alias และ interface
  • Generics และ Utility Types สำเร็จรูป เช่น Readonly, Partial, Pick, Omit

Workflow และ Ecosystem

  • ตั้งค่า Compiler ด้วย tsconfig.json และ Base สำเร็จรูปจากชุมชน
  • ย้ายโปรเจกต์ JavaScript เดิมมาเป็น TypeScript ได้ทีละไฟล์ ไม่ต้องเขียนใหม่ทั้งหมด
  • ใช้ @types/ เพื่อได้ Type ของ Library ที่ไม่ได้เขียนด้วย TypeScript
  • แชร์ Type ระหว่างไฟล์และระหว่าง Frontend/Backend เพื่อ End-to-End Type Safety
  • Library อย่าง Zod และ TypeORM ขยายประโยชน์ของ Type ไปถึงข้อมูลตอนรันจริงและฐานข้อมูล

คำถามชวนคิด

ลองตอบคำถามเหล่านี้ด้วยตัวเองก่อนไปเรียนหัวข้อถัดไป เพื่อเช็กว่าภาพรวมของ TypeScript ในหัวเราแน่นแค่ไหน:

  1. มุมมองต่อ TypeScript ของเราเปลี่ยนไปอย่างไรบ้างเทียบกับตอนก่อนเริ่มสัปดาห์นี้?
  2. Feature ไหนของ TypeScript ที่เรารู้สึกว่ามีประโยชน์กับงานที่เคยเขียนมากที่สุด?
  3. มี Feature ไหนที่ยังรู้สึกสับสน หรืออยากทำความเข้าใจให้ลึกขึ้นอีก?

แหล่งเรียนรู้เพิ่มเติม

  • TypeScript Handbook — เอกสารทางการ อ่านฟรี ครอบคลุมตั้งแต่พื้นฐานถึงขั้นสูง
  • TypeScript Docs — คู่มืออ้างอิงฉบับสมบูรณ์ของทีม TypeScript เอง
  • TypeScript Playground — เว็บสำหรับทดลองเขียน TypeScript และดูผลลัพธ์ที่ Compile ออกมาเป็น JavaScript ได้ทันทีโดยไม่ต้องติดตั้งอะไร
  • typescript-first-steps โดย Anjana Vakil — ต้นทางแรงบันดาลใจของบทเรียนชุดนี้ มีแบบฝึกหัดเชิงลึกให้ลองทำเพิ่มเติม
  • awesome-typescript — รวมรายชื่อ Library และแหล่งข้อมูล TypeScript ยอดนิยมจากชุมชน

สิ่งสำคัญที่สุดในการเรียนรู้ TypeScript ต่อจากนี้คือการลงมือใช้จริง — ลองเปิดโปรเจกต์เก่าของตัวเอง แล้วค่อย ๆ ย้ายมาเป็น TypeScript ทีละไฟล์ตามขั้นตอนที่เราเรียนไปในบทที่แล้ว จะเป็นวิธีที่ช่วยให้ความเข้าใจแน่นขึ้นเร็วที่สุด
