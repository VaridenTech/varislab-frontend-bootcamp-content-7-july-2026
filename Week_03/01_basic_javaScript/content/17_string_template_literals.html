การใช้งาน String Template Literals

ที่ผ่านมาเราได้สร้างข้อมูลข้อความ (String) โดยใช้เครื่องหมายคำพูดเดี่ยว '' (Single Quotes) หรือเครื่องหมายคำพูดคู่ "" (Double Quotes) แต่ในภาษา JavaScript ยังมีวิธีที่สามซึ่งมีความยืดหยุ่นและทรงพลังกว่ามาก นั่นคือการใช้ Template Literals ด้วยสัญลักษณ์ Backtick ``

โครงสร้างไวยากรณ์พื้นฐาน (Syntax)

Template Literals จะใช้สัญลักษณ์ Backtick `` แทนเครื่องหมายคำพูดปกติ (ซึ่งบนคีย์บอร์ดภาษาไทยมักจะเป็นปุ่มเดียวกับปุ่มเปลี่ยนภาษา หรือปุ่ม Grave Accent ที่อยู่มุมซ้ายบนสุดเหนือปุ่ม Tab):

const a = `Hi`
console.log(a) // Hi

ในแง่การใช้งานทั่วไปจะแสดงผลเหมือน String ปกติทุกประการ แต่ความพิเศษของ Template Literals คือฟีเจอร์ที่อยู่ด้านล่างนี้:

1. การฝังตัวแปรลงในข้อความ (String Interpolation)

จุดเด่นที่สำคัญที่สุดของ Template Literals คือเราสามารถนำตัวแปรมาแทรกหรือฝังไว้ภายในข้อความได้โดยตรงผ่านสัญลักษณ์ ${ตัวแปร} โดยไม่จำเป็นต้องใช้เครื่องหมายบวก (+) เพื่อต่อข้อความให้ยุ่งยากอีกต่อไป:

const firstName = "Kyle"
const lastName = "Cook"

console.log(firstName + " " + lastName) // Kyle Cook
console.log(`${firstName} ${lastName}`) // Kyle Cook

สัญลักษณ์ ${} จะทำหน้าที่บอกให้ JavaScript Engine ทราบว่า "ให้ประมวลผลโค้ดที่อยู่ภายในวงเล็บปีกกานี้ แล้วนำผลลัพธ์ที่ได้มาแปลงเป็นข้อความแทรกลงในตำแหน่งนี้ทันที"

2. การฝังนิพจน์หรือการคำนวณ (Expressions)

นอกจากตัวแปรแล้ว เรายังสามารถเขียนคำสั่งคำนวณทางคณิตศาสตร์ เรียกใช้ Function หรือใส่ตรรกะใด ๆ ของ JavaScript ลงไปในสัญลักษณ์ ${} ได้ด้วย:

const firstName = "Kyle"

console.log(`${firstName} ${2 + 3}`) // Kyle 5
console.log(`Hello ${firstName.toUpperCase()}`) // Hello KYLE
console.log(`Random number: ${Math.random()}`) // Random number: 0.234...

3. การเขียนข้อความแบบหลายบรรทัด (Multi-line Strings)

หากเป็นเครื่องหมายคำพูดแบบปกติ การเขียนข้อความขึ้นบรรทัดใหม่จะทำได้ยากและต้องใช้สัญลักษณ์พิเศษอย่าง \n แต่สำหรับ Template Literals เราสามารถกดปุ่ม Enter เพื่อขึ้นบรรทัดใหม่ในโค้ดได้ตรง ๆ เลย:

const poem = `Roses are red,
Violets are blue,
Template literals are awesome,
And so are you!`

console.log(poem)
// Roses are red,
// Violets are blue,
// Template literals are awesome,
// And so are you!

สรุปแนวทางการใช้งาน

ควรเลือกใช้ Template Literals เมื่อ:

  • ต้องการแทรกตัวแปรหรือคำนวณค่าต่าง ๆ ลงในข้อความ
  • ต้องการเขียนข้อความที่มีการขึ้นบรรทัดใหม่ (เช่น โครงสร้าง HTML)

ควรเลือกใช้เครื่องหมายคำพูดปกติ ('' หรือ "") เมื่อ:

  • เป็นข้อความสั้น ๆ ทั่วไปที่ไม่มีการแทรกตัวแปรหรือการคำนวณใด ๆ เพื่อให้อ่านโค้ดได้ง่ายและเป็นระเบียบ
