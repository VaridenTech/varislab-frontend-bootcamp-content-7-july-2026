การใช้งาน Window และ Document Object

เมื่อเรานำ JavaScript ไปรันบนเว็บ Browser ตัว Browser จะเตรียม Object ระดับโกลบอล (Global Objects) พื้นฐานมาให้เราใช้งาน 2 ตัวหลัก ๆ คือ window และ document ซึ่งทำหน้าที่เป็นสะพานเชื่อมต่อระหว่างโค้ด JavaScript กับหน้าต่าง Browser และหน้าเว็บ HTML ของเรา

1. ทำความเข้าใจเกี่ยวกับ Window Object

Object window คือตัวแทนของ หน้าต่างเว็บ Browser (Browser Window) และเป็น Object ระดับสูงสุด (Global Object) ของระบบ ทุก ๆ ตัวแปร Function หรือเครื่องมือสำเร็จรูปที่เราเรียกใช้งานแบบลอย ๆ (Global Scope) แท้จริงแล้วล้วนเป็นคุณสมบัติ (Properties) หรือ Method (Methods) ที่อยู่ภายใต้ window ทั้งสิ้น:

คุณสมบัติความเป็น Global ของ Window

// ทั้งสองคู่ด้านล่างนี้ทำงานเหมือนกันทุกประการ
console.log("Hello")
window.console.log("Hello")

alert("Hi there!")
window.alert("Hi there!")

ดังนั้น เมื่อใดก็ตามที่เราเรียกใช้ Function ระดับ Global ทาง JavaScript Engine จะเข้าใจโดยอัตโนมัติว่าเรากำลังอ้างอิงถึง window.ชื่อ Function เสมอ

การทดลองสำรวจ Window Object

console.log(window)

หากคุณลองพิมพ์คำสั่งด้านบนใน Console ของเครื่องมือผู้พัฒนา (DevTools) คุณจะพบกับรายการคุณสมบัติและ Method จำนวนมหาศาลที่ Browser เตรียมมาให้ (เช่น ข้อมูลขนาดหน้าจอ ประวัติการเข้าเว็บ หรือการตั้งเวลา) ซึ่งไม่จำเป็นต้องท่องจำทั้งหมด เลือกศึกษาเฉพาะตัวที่ต้องใช้งานก็พอครับ

กรณีที่เราจำเป็นต้องระบุ Keyword window เจาะจง

  1. 1. เพื่อหลีกเลี่ยงปัญหาชื่อตัวแปรชนกัน (Name Collision)

    const alert = "This is my custom alert message"
// alert("Hello") // ❌ Error เพราะระบบจะมองเป็นตัวแปร string ด้านบนแทน Function
window.alert(alert) // ✅ ระบุ window นำหน้าเพื่อเรียกใช้ Function ดั้งเดิมของ Browser

  2. 2. เมื่อต้องการประกาศตัวแปร Global เจาะจง

    window.myGlobalVar = "Hello, World!"
console.log(myGlobalVar) // ✅ "Hello, World!"

    💡 แนวทางปฏิบัติที่ดี: ไม่แนะนำให้สร้างตัวแปร Global ขึ้นมาเองผ่าน window หากไม่จำเป็นจริง ๆ เพราะอาจทำให้ตัวแปรไปทับซ้อนกับไลบรารีอื่น ๆ และทำให้ตรวจสอบหา Bug ได้ยากขึ้น

2. ทำความเข้าใจเกี่ยวกับ Document Object (DOM)

Object document คือตัวแทนของ หน้าเอกสาร HTML (HTML Document) ที่กำลังแสดงผลอยู่ในหน้าต่าง Browser ขณะนั้น โครงสร้าง Object นี้เรามักจะคุ้นเคยกันในชื่อ DOM (Document Object Model) ซึ่งช่วยให้ JavaScript สามารถเข้าไปอ่าน แก้ไข เพิ่มลบ หรือตกแต่งหน้าเว็บ HTML ได้ตามต้องการ

ความสัมพันธ์ระหว่าง Window และ Document

console.log(document)

หากคุณลองรันคำสั่งด้านล่าง จะพบว่า document ก็คือคุณสมบัติ (Property) หนึ่งที่อยู่ภายใต้ Object window นั่นเอง (เขียนย่อเหลือเพียง document ได้เพราะ window เป็น Global):

console.log(document)
console.log(window.document)

คุณสมบัติ (Properties) ที่น่าสนใจของ Document

console.log(document.documentElement) // Element <html>
console.log(document.body) // Element <body>
console.log(document.head) // Element <head>
console.log(document.URL) // แสดง URL ของหน้าเว็บปัจจุบัน
