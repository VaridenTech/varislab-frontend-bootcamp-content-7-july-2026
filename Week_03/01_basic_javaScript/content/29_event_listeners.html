Event Listener

Event listener ทำให้หน้าเว็บโต้ตอบได้ โดยตอบสนองต่อการคลิก การพิมพ์ การส่งฟอร์ม และอื่น ๆ

Event คืออะไร?

Event คือสิ่งที่เกิดขึ้นในเบราว์เซอร์ เช่น ผู้ใช้คลิกปุ่ม พิมพ์ในช่องข้อมูล ส่งฟอร์ม หรือปรับขนาดหน้าต่าง Event listener คือฟังก์ชันที่ “ฟัง” Event เหล่านี้และรันโค้ดเมื่อเกิดขึ้น

ไวยากรณ์พื้นฐาน

const button = document.querySelector("button")
button.addEventListener("click", () => {
  console.log("Button was clicked!")
})

addEventListener รับสองอาร์กิวเมนต์หลัก:

  • eventType: String ระบุชนิด Event เช่น “click”, “input”, “submit”
  • function: ฟังก์ชันที่จะรันเมื่อเกิด Event

การเพิ่ม Event Listener ให้ Element

const input = document.querySelector("input")
input.addEventListener("input", (e) => {
  console.log("Input value:", e.target.value)
})
document.addEventListener("click", () => console.log("Document was clicked"))
window.addEventListener("resize", () => {
  console.log("Window resized to:", window.innerWidth, window.innerHeight)
})

Event Listener หลายตัว

เพิ่ม Listener หลายตัวให้ Element เดียวกันได้ โดยไม่เขียนทับกันและทำงานตามลำดับที่เพิ่ม:

button.addEventListener("click", () => console.log("First listener"))
button.addEventListener("click", () => console.log("Second listener"))

การลบ Event Listener

ต้องมี Reference ของฟังก์ชันตัวเดิม:

function handleClick() { console.log("Clicked!") }
button.addEventListener("click", handleClick)
button.removeEventListener("click", handleClick)
// ❌ ฟังก์ชันคนละ Object จึงลบไม่ได้
button.addEventListener("click", () => console.log("Clicked!"))
button.removeEventListener("click", () => console.log("Clicked!"))

Event Object

ฟังก์ชัน Listener ได้รับ Event object ที่มีข้อมูลเกี่ยวกับ Event โดยอัตโนมัติ:

button.addEventListener("click", (event) => console.log(event))

พร็อพเพอร์ตีที่พบบ่อย

button.addEventListener("click", (e) => {
  console.log(e.target)
  console.log(e.currentTarget)
  console.log(e.clientX, e.clientY)
  console.log(e.ctrlKey, e.shiftKey, e.altKey)
})

target กับ currentTarget

  • target - Element จริงที่ทำให้เกิด Event อาจเป็น Element ลูก
  • currentTarget - Element ที่ติด Listener ไว้

โดยทั่วไป currentTarget คือค่าที่ต้องการเมื่อทำงานใน Listener

ชนิด Event ที่พบบ่อย

Mouse

  • click, dblclick, mouseenter, mouseleave, mouseover

Focus

  • focus, blur

Input

  • input - ทำงานทุกครั้งที่ค่าเปลี่ยน
  • change - ทำงานเมื่อเสีย Focus และค่าเปลี่ยน

Window

  • resize - เมื่อขนาดหน้าต่างเปลี่ยน

ป้องกันพฤติกรรมเริ่มต้นด้วย preventDefault

Event หลายชนิดมีพฤติกรรมเริ่มต้น เช่น ลิงก์นำทางและฟอร์มเปลี่ยนหน้า:

const form = document.querySelector("form")
form.addEventListener("submit", (e) => {
  e.preventDefault()
  const formData = new FormData(e.currentTarget)
  console.log("Username:", formData.get("username"))
})
link.addEventListener("click", (e) => e.preventDefault())
document.addEventListener("contextmenu", (e) => e.preventDefault())

ข้อผิดพลาดที่พบบ่อย

  1. เรียกฟังก์ชันโดยไม่ตั้งใจ

    // ❌ ผิด
button.addEventListener("click", handleClick())
// ✅ ถูกต้อง
button.addEventListener("click", handleClick)

  2. ใช้ target แทน currentTarget

    button.addEventListener("click", (e) => {
  console.log(e.target) // อาจเป็น Element ลูก
  console.log(e.currentTarget) // เป็นปุ่มเสมอ
})
