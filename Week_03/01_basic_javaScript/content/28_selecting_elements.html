การเลือก Element

ทักษะพื้นฐานของการพัฒนาเว็บคือการเลือก Element HTML เพื่อโต้ตอบด้วย JavaScript โดยมีหลายเมธอดที่เหมาะกับกรณีต่างกัน

วิธีที่ 1: getElementById

เลือก Element เดียวจากแอตทริบิวต์ ID

<div id="header">This is a header</div>
const header = document.getElementById("header")
console.log(header)
header.style.color = "blue"

ประเด็นสำคัญ:

  • ส่ง Element เดียวกลับมา หรือ null หากไม่พบ
  • ID ควรไม่ซ้ำในหน้า
  • ส่งเพียงชื่อ ID ไม่ใช่ #header
  • รวดเร็วและมีประสิทธิภาพ
// ❌ ผิด
document.getElementById("#header")
// ✅ ถูกต้อง
document.getElementById("header")

วิธีที่ 2: getElementsByClassName

เลือกหลาย Element จากชื่อ Class

<div class="content">First</div>
<div class="content">Second</div>
<div class="content">Third</div>
const contentDivs = document.getElementsByClassName("content")
console.log(contentDivs.length) // 3
console.log(contentDivs[0])

การทำงานกับ HTMLCollection

HTMLCollection ดูคล้าย Array แต่ไม่มีเมธอด Array ทั้งหมด:

// แปลงเป็น Array ก่อน
Array.from(contentDivs).forEach((div) => div.style.color = "green")

// หรือใช้ For loop
for (let i = 0; i < contentDivs.length; i++) {
  contentDivs[i].style.backgroundColor = "lightgray"
}
contentDivs[0].style.fontWeight = "bold"

วิธีที่ 3: querySelector (แนะนำ)

ใช้ CSS selector และส่ง Element ตัวแรก ที่ตรงกันกลับมา:

const header = document.querySelector("#header")
const firstContent = document.querySelector(".content")
const firstDiv = document.querySelector("div")
const submitButton = document.querySelector('[data-action="submit"]')

วิธีที่ 4: querySelectorAll (แนะนำ)

ส่ง Element ทั้งหมด ที่ตรงกันกลับมาเป็น NodeList:

const allContent = document.querySelectorAll(".content")
const allDivs = document.querySelectorAll("div")
const allInputs = document.querySelectorAll("input")

การทำงานกับ NodeList

NodeList คล้าย Array และมี forEach:

document.querySelectorAll(".content").forEach((div) => {
  div.style.padding = "10px"
})

Collection แบบ Live กับ Static

getElementsByClassName ส่ง HTMLCollection แบบ Live ซึ่งอัปเดตเมื่อ DOM เปลี่ยน ส่วน querySelectorAll ส่ง NodeList แบบ Static ซึ่งไม่เปลี่ยนจนกว่าจะเรียกใหม่

const liveCollection = document.getElementsByClassName("content")
const newDiv = document.createElement("div")
newDiv.className = "content"
document.body.appendChild(newDiv)
console.log(liveCollection.length) // อัปเดตอัตโนมัติ
const staticNodeList = document.querySelectorAll(".content")
// หลังเพิ่ม Element ค่า length ยังคงเดิม

ส่วนใหญ่ไม่ต้องการ Collection แบบ Live เพราะทำให้โค้ดเข้าใจยากและคาดเดาได้น้อยลง

ควรใช้เมธอดใด?

โดยค่าเริ่มต้นควรเลือก querySelector และ querySelectorAll เพราะยืดหยุ่น ใช้ CSS selector ที่คุ้นเคย และได้ NodeList แบบ Static ใช้ getElementById เมื่อต้องเลือก Element เดียวด้วย ID และต้องการประสิทธิภาพสูงสุด

การใช้ CSS Selector ขั้นสูง

document.querySelector('input[name="username"]')
document.querySelectorAll('a[href^="https://"]')
document.querySelector("li:first-child")
document.querySelector("div:not(.hidden)")
document.querySelector("nav > ul > li")
document.querySelector("h2 + p")
document.querySelector("article p")
document.querySelector("button, input[type='submit']")

แบบฝึกหัด

เปลี่ยนข้อความ Header เป็น “Welcome!”, ทำ Paragraph ทั้งหมดเป็นสีน้ำเงิน, ซ่อน Paragraph สุดท้าย และเพิ่มเส้นขอบให้ Input ที่มี Class “required”

เฉลย
document.querySelector("#main-header").textContent = "Welcome!"
document.querySelectorAll("p").forEach((p) => p.style.color = "blue")
document.querySelector("p:last-child").style.display = "none"
document.querySelectorAll("input.required").forEach((input) => {
  input.style.border = "2px solid red"
})
