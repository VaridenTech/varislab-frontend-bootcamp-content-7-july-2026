เขียนโปรแกรม JavaScript แรกของคุณ

มาเริ่มต้นเขียนโปรแกรม JavaScript แรกกันเลย! เราจะสร้างโปรแกรม "Hello World" ซึ่งเป็นโปรแกรมเริ่มต้นยอดนิยมของนักเขียนโค้ดทั่วโลก

การเตรียมพื้นที่ทำงาน (Workspace Setup)

  1. เปิดโปรแกรม VS Code
  2. สร้างโฟลเดอร์สำหรับเก็บโปรเจกต์
    • คลิกเมนู File → Open Folder
    • เลือกหรือสร้างโฟลเดอร์ใหม่สำหรับโปรเจกต์ของคุณ

โครงสร้างหน้าต่างโปรแกรม VS Code

เมื่อเปิดโฟลเดอร์ขึ้นมาแล้ว คุณจะพบกับส่วนประกอบหลักดังนี้:

  • แถบด้านซ้าย: แถบสำรวจไฟล์ (File Explorer)
  • พื้นที่ตรงกลาง: พื้นที่เขียนโค้ด (Editor)
  • แถบด้านล่าง: Terminal (Terminal - สำหรับใช้รันและทดสอบโค้ด)

คีย์ลัด (Shortcuts) ที่จำเป็น

  • เปิด/ปิดแถบด้านข้าง: Ctrl + B
  • เปิด/ปิด Terminal: Ctrl + ~
  • บันทึกไฟล์ (Save): Ctrl + S

ขั้นตอนการเขียนโค้ด JavaScript แรก

1. สร้างไฟล์ใหม่

  • คลิกปุ่ม New File ในแถบด้านข้าง
  • ตั้งชื่อไฟล์ว่า: script.js
  • นามสกุล .js จะช่วยให้ VS Code ทราบว่าไฟล์นี้เป็นไฟล์ JavaScript

2. เขียนโค้ดแรกของคุณ

พิมพ์โค้ดต่อไปนี้ลงในไฟล์ script.js:

console.log("Hello World")

3. บันทึกไฟล์

  • จุดสีขาวข้างชื่อไฟล์จะหายไปเมื่อคุณบันทึกไฟล์เรียบร้อยแล้ว

วิธีการรันโค้ด

Node.js

  1. เปิด Terminal (Terminal)
  2. พิมพ์คำสั่ง node script.js แล้วกด Enter

ผลลัพธ์: คุณจะพบข้อความ "Hello World" แสดงขึ้นมาบน Terminal!

เกร็ดความรู้เพิ่มเติม

  • คุณสามารถกดปุ่ม ลูกศรขึ้น (Up Arrow) เพื่อเรียกใช้คำสั่งล่าสุดอีกครั้งได้อย่างรวดเร็ว

เว็บ Browser (Web Browser)

  1. สร้างไฟล์ HTML ใหม่ชื่อ index.html 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My First App</title>
    <script src="script.js"></script>
  </head>
  <body>
    <h1>My First JavaScript App</h1>
  </body>
</html>
  2. เปิดไฟล์ด้วย Live Server
  3. ตรวจสอบผลลัพธ์การทำงานของ JavaScript
    • คลิกขวาบนพื้นที่ว่างของหน้าเว็บ
    • เลือกเมนู "Inspect" (หรือกดปุ่มลัด Ctrl + Shift + I)
    • คลิกไปที่แท็บ "Console"
    • คุณจะพบข้อความ "Hello World" แสดงอยู่ตรงนั้น!

ข้อผิดพลาดที่พบบ่อย (Common Errors)

ลืมใส่เครื่องหมายคำพูด (Quotes)

// ❌ ผิด
console.log(Hello World);

// ✅ ถูกต้อง
console.log('Hello World');

ลืมใส่วงเล็บ (Parentheses)

// ❌ ผิด
console.log'Hello World';

// ✅ ถูกต้อง
console.log('Hello World');

สะกดคำสั่ง console.log ผิด

// ❌ ผิด
consol.log("Hello World")

// ✅ ถูกต้อง
console.log("Hello World")

การเขียน Comment (Comments) ใน JavaScript

Comment คือข้อความอธิบายโค้ดที่เขียนขึ้นเพื่อให้มนุษย์อ่านเข้าใจง่ายขึ้น โดย JavaScript Engine จะข้ามการทำงานและไม่นำ Comment เหล่านี้ไปประมวลผล

การเขียน Comment บรรทัดเดียว (Single-line Comment)

ใช้เครื่องหมาย // สำหรับ Comment ข้อความสั้น ๆ ในบรรทัดเดียว:

// นี่คือ Comment บรรทัดเดียว
console.log("Hello World") // คำสั่งนี้จะแสดง "Hello World"

การเขียน Comment หลายบรรทัด (Multi-line Comment)

ใช้เครื่องหมาย /* ... */ สำหรับ Comment ข้อความที่มีความยาวหลายบรรทัด: 

/*
  นี่คือ Comment หลายบรรทัด
  ซึ่งสามารถครอบคลุมได้หลายบรรทัด
*/
console.log("Hello World")
