แอป JavaScript แรกของคุณ

มาเขียนโปรแกรม JavaScript โปรแกรมแรกของคุณกัน! เราจะสร้างแอปพลิเคชัน "Hello World" แบบคลาสสิก

การตั้งค่าพื้นที่ทำงานของคุณ

  1. เปิด VSCode
  2. สร้างโฟลเดอร์โปรเจกต์
    • คลิก File → Open Folder
    • เลือกหรือสร้างโฟลเดอร์ใหม่สำหรับโปรเจกต์ของคุณ

หน้าตาของ VSCode

เมื่อเปิดโฟลเดอร์แล้ว คุณจะเห็น:

  • แถบด้านซ้าย: ตัวสำรวจไฟล์
  • ตรงกลาง: ตัวแก้ไขโค้ด
  • ด้านล่าง: เทอร์มินัล (เราจะใช้ส่วนนี้เพื่อรันโค้ด)

แป้นพิมพ์ลัดที่สำคัญ

  • แสดงหรือซ่อนแถบด้านข้าง: Ctrl + B
  • แสดงหรือซ่อนเทอร์มินัล: Ctrl + ~
  • บันทึกไฟล์: Ctrl + S

การสร้างแอป JavaScript แรกของคุณ

1. สร้างไฟล์

  • คลิกปุ่ม New File ในแถบด้านข้าง
  • ตั้งชื่อว่า: script.js
  • นามสกุล .js จะบอก VSCode ว่าไฟล์นี้เป็น JavaScript

2. เขียนโค้ดแรกของคุณ

ในไฟล์ script.js ให้พิมพ์ตามนี้ทุกประการ:

console.log("Hello World")

3. บันทึกไฟล์

  • จุดสีขาวข้างชื่อไฟล์จะหายไปเมื่อบันทึกไฟล์แล้ว

การรันโค้ดของคุณ

Node.js

  1. เปิดเทอร์มินัล
  2. รันคำสั่ง node script.js

ผลลัพธ์: คุณจะเห็น "Hello World" แสดงอยู่ในเทอร์มินัล!

เคล็ดลับสั้น ๆ

  • กด ลูกศรขึ้น เพื่อรันคำสั่งล่าสุดอีกครั้ง

เบราว์เซอร์

  1. สร้างไฟล์ HTML ชื่อ index.html 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My First App</title>
    <script src="script.js"></script>
  </head>
  <body>
    <h1>My First JavaScript App</h1>
  </body>
</html>
  2. เปิดด้วย Live Server
  3. ดูผลลัพธ์จาก JavaScript
    • คลิกขวาบนหน้าเว็บ
    • เลือก "Inspect" (หรือกด Ctrl + Shift + I)
    • คลิกแท็บ "Console"
    • คุณจะเห็น "Hello World" แสดงอยู่ตรงนั้น!

ข้อผิดพลาดที่พบบ่อย

ลืมใส่เครื่องหมายคำพูด

// ❌ ผิด
console.log(Hello World);

// ✅ ถูกต้อง
console.log('Hello World');

ลืมใส่วงเล็บ

// ❌ ผิด
console.log'Hello World';

// ✅ ถูกต้อง
console.log('Hello World');

พิมพ์ console.log ผิด

// ❌ ผิด
consol.log("Hello World")

// ✅ ถูกต้อง
console.log("Hello World")

คอมเมนต์ใน JavaScript

คุณสามารถเพิ่มคอมเมนต์ลงในโค้ดเพื่อให้เข้าใจได้ง่ายขึ้น เอนจิน JavaScript จะไม่ประมวลผลคอมเมนต์เหล่านี้

คอมเมนต์บรรทัดเดียว

ใช้ // เพื่อเพิ่มคอมเมนต์ในบรรทัดเดียว:

// นี่คือคอมเมนต์บรรทัดเดียว
console.log("Hello World") // คำสั่งนี้จะแสดง "Hello World"

คอมเมนต์หลายบรรทัด

ใช้ /* ... */ เพื่อเพิ่มคอมเมนต์ที่ครอบคลุมหลายบรรทัด: 

/*
  นี่คือคอมเมนต์หลายบรรทัด
  ซึ่งสามารถครอบคลุมได้หลายบรรทัด
*/
console.log("Hello World")
