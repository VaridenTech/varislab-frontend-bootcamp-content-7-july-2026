การใช้งานเงื่อนไข If Else

คำสั่งเงื่อนไข If Else เป็นหนึ่งในแนวคิดพื้นฐานและสำคัญที่สุดของการเขียนโปรแกรม ช่วยให้เราสามารถควบคุมทิศทางการทำงานของโค้ด (Control Flow) ให้ทำงานแตกต่างกันออกไปตามเงื่อนไขที่เรากำหนดว่าเป็นจริง (True) หรือเท็จ (False)

1. คำสั่ง If พื้นฐาน (Basic If Statement)

รูปแบบที่เรียบง่ายที่สุดคือการตรวจสอบเงื่อนไข หากเงื่อนไขในวงเล็บเป็นจริง (True) ชุดคำสั่งภายในวงเล็บปีกกาจะทำงานทันที แต่ถ้าเป็นเท็จ (False) ระบบจะข้ามไปทำงานในส่วนถัดไป:

const age = 18

if (age >= 18) {
  console.log("You can vote!") // บรรทัดนี้จะทำงาน
}

console.log("Program continues...") // บรรทัดนี้ทำงานเสมอ

โครงสร้างไวยากรณ์ประกอบด้วย:

  • Keyword if
  • เงื่อนไขที่ต้องการตรวจสอบภายในวงเล็บ ()
  • บล็อกคำสั่งที่ต้องการให้ทำงานภายในวงเล็บปีกกา {}

2. คำสั่ง If...Else (กรณีมีทางเลือกอื่น)

เราสามารถใช้ Keyword else เพื่อกำหนดชุดคำสั่งที่จะทำงานในกรณีที่เงื่อนไขแรกใน if เป็นเท็จ (False):

const age = 16

if (age >= 18) {
  console.log("You can vote!")
} else {
  console.log("You cannot vote yet") // บรรทัดนี้จะทำงาน
}

3. การตรวจสอบหลายเงื่อนไขด้วย Else If

หากเรามีเงื่อนไขที่ต้องการตรวจสอบมากกว่าสองทางเลือก เราสามารถใช้ else if เพื่อตรวจสอบเงื่อนไขอื่น ๆ ไล่ตามลำดับลงมาได้:

const temperature = 75

if (temperature > 80) {
  console.log("It's hot outside!")
} else if (temperature > 60) {
  console.log("Perfect weather!") // บรรทัดนี้จะทำงาน
} else if (temperature > 40) {
  console.log("It's a bit chilly")
} else {
  console.log("It's cold!")
}

⚠️ ข้อควรจำ: JavaScript จะทำงานเฉพาะในบล็อกเงื่อนไข แรกสุดที่เป็นจริงเท่านั้น เมื่อพบเงื่อนไขที่เป็นจริงแล้ว ระบบจะข้ามเงื่อนไขที่เหลือทั้งหมดด้านล่างทันที แม้ว่าเงื่อนไขอื่น ๆ ด้านล่างจะเป็นจริงด้วยก็ตาม

4. ตัวดำเนินการทางตรรกศาสตร์ (Logical Operators)

เราสามารถใช้ตัวดำเนินการทางตรรกศาสตร์เพื่อเชื่อมหรือรวมเงื่อนไขหลาย ๆ ข้อเข้าด้วยกันได้:

AND (&&) - ทุกเงื่อนไขต้องเป็นจริงพร้อมกัน

const age = 25
const hasLicense = true

if (age >= 18 && hasLicense) {
  console.log("You can drive!") // ทั้งสองเงื่อนไขเป็นจริง
}

OR (||) - ขอเพียงเงื่อนไขใดเงื่อนไขหนึ่งเป็นจริงก็พอ

const isWeekend = false
const isHoliday = true

if (isWeekend || isHoliday) {
  console.log("No work today!") // หนึ่งเงื่อนไขเป็นจริง
}

NOT (!) - กลับค่าความจริง (จากจริงเป็นเท็จ จากเท็จเป็นจริง)

const isLoggedIn = false

if (!isLoggedIn) {
  console.log("Please log in") // ทำงานเพราะ !false คือ true
}

5. ค่า Truthy และ Falsy

ในภาษา JavaScript ตัวแปรหรือค่าข้อมูลทั่วไปที่ไม่ใช่ Boolean (Boolean) ก็สามารถนำมาตรวจสอบในเงื่อนไข if ได้ โดยระบบจะมองค่าเหล่านั้นเป็น Truthy (เทียบเท่าจริง) หรือ Falsy (เทียบเท่าเท็จ) ตามกฎของภาษา:

ค่า Falsy (ค่าที่ถูกมองว่าเป็น false):

  • false
  • 0
  • "" (String ว่าง)
  • null
  • undefined
  • NaN

ค่า Truthy (ค่าที่ถูกมองว่าเป็น true):

  • ค่าข้อมูลอื่น ๆ ทั้งหมดนอกเหนือจาก Falsy (รวมถึงข้อความที่มีตัวเลข "0", Array ว่าง [], และ Object ว่าง {})
const name = "Kyle"

if (name) {
  console.log(`Hello ${name}!`) // ทำงานเพราะ String ที่ไม่ว่างเป็น truthy
}

const emptyString = ""
if (emptyString) {
  console.log("This won't run") // String ว่างเป็น falsy
}

6. คำสั่ง If ซ้อนกัน (Nested If)

เราสามารถเขียนคำสั่ง if ซ้อนอยู่ภายในบล็อก if อื่นได้ตามต้องการ:

const weather = "sunny"
const temperature = 75

if (weather === "sunny") {
  if (temperature > 70) {
    console.log("Perfect day for the beach!") // บรรทัดนี้จะทำงาน
  } else {
    console.log("Sunny but a bit cold")
  }
} else {
  console.log("Not a sunny day")
}

💡 คำแนะนำ: ในทางปฏิบัติไม่แนะนำให้เขียนโค้ดซ้อนกันหลาย ๆ ชั้น (Nested If) เพราะจะทำให้โค้ดอ่านยากและซับซ้อนต่อการดูแลรักษา

ทางเลือกที่ดีกว่า: การใช้ Guard Clauses

Guard Clause คือเทคนิคการเขียนโค้ดเพื่อตรวจสอบและจัดการเงื่อนไขที่ไม่ถูกต้องหรือกรณีข้อยกเว้นไว้ที่ด้านบนสุดของ Function แล้วสั่งให้หยุดทำงานทันที (ด้วย return) ช่วยลดการเขียนโค้ดซ้อนกันหลายชั้น ทำให้โค้ดสะอาดและอ่านง่ายขึ้นมาก:

const weather = "sunny"
const temperature = 75

if (weather !== "sunny") {
  console.log("Not a sunny day")
  return // ออกจากการทำงานก่อน
}

if (temperature > 70) {
  console.log("Perfect day for the beach!") // บรรทัดนี้จะทำงาน
} else {
  console.log("Sunny but a bit cold")
}

7. การเขียนคำสั่ง If แบบบรรทัดเดียว (Shorthand If)

หากชุดคำสั่งภายในบล็อก if หรือ else มีเพียงบรรทัดเดียว เราสามารถละเว้น (ไม่ต้องเขียน) วงเล็บปีกกา {} ได้เพื่อความกระชับ:

const age = 18
if (age >= 18) console.log("You can vote!") // บรรทัดนี้จะทำงาน
else console.log("You cannot vote yet") // บรรทัดนี้จะไม่ทำงาน

💡 แนวทางปฏิบัติที่ดี: แนะนำให้ใช้รูปแบบนี้เฉพาะกับคำสั่งสั้น ๆ เช่น Guard Clause เท่านั้น หากมีคำสั่งหลายบรรทัดหรือซับซ้อน ควรใส่วงเล็บปีกกา {} เสมอเพื่อความปลอดภัยและป้องกันข้อผิดพลาด

ลองทำแบบฝึกหัด

ฝึกฝนความเข้าใจด้วยการเขียนเงื่อนไขตรวจสอบคุณสมบัติของผู้ขอใบขับขี่ ดังนี้:

  1. เงื่อนไข: จะต้องมีอายุอย่างน้อย 16 ปี และผ่านการอบรมหลักสูตรขับขี่เรียบร้อยแล้ว
  2. หากผ่านเงื่อนไขทั้งหมด ให้แสดงข้อความ “You can get your license!”
  3. หากอายุไม่ถึง 16 ปี ให้แสดงข้อความ “You must be at least 16”
  4. หากอายุถึงแต่ยังไม่อบรมหลักสูตรขับขี่ ให้แสดงข้อความ “You need to complete driver's education”
ดูเฉลย 
const age = 17
const hasDriversEd = true

if (age < 16) {
  console.log("You must be at least 16")
} else if (!hasDriversEd) {
  console.log("You need to complete driver's education")
} else {
  console.log("You can get your license!")
}

อีกวิธีหนึ่งที่สะอาดกว่าโดยใช้เทคนิค Guard Clause:

const age = 17
const hasDriversEd = true

if (age < 16) {
  console.log("You must be at least 16")
  return
}

if (!hasDriversEd) {
  console.log("You need to complete driver's education")
  return
}

console.log("You can get your license!")
