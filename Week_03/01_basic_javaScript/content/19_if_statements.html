คำสั่ง If

คำสั่ง If เป็นหนึ่งในแนวคิดพื้นฐานที่สุดของการเขียนโปรแกรม ช่วยให้คุณรันโค้ดต่างกันตามว่าเงื่อนไขบางอย่างเป็นจริงหรือเท็จ

คำสั่ง If พื้นฐาน

รูปแบบที่ง่ายที่สุดของคำสั่ง If จะตรวจสอบเงื่อนไขและรันโค้ดเฉพาะเมื่อเงื่อนไขเป็นจริง:

const age = 18

if (age >= 18) {
  console.log("You can vote!") // บรรทัดนี้จะทำงาน
}

console.log("Program continues...") // บรรทัดนี้ทำงานเสมอ

ไวยากรณ์ประกอบด้วย:

  • คีย์เวิร์ด if
  • เงื่อนไขในวงเล็บ ()
  • บล็อกโค้ดในวงเล็บปีกกา {}

คำสั่ง If...Else

ใช้ else เพื่อระบุว่าจะเกิดอะไรขึ้นเมื่อเงื่อนไขเป็นเท็จ:

const age = 16

if (age >= 18) {
  console.log("You can vote!")
} else {
  console.log("You cannot vote yet") // บรรทัดนี้จะทำงาน
}

หลายเงื่อนไขด้วย Else If

ใช้ else if เพื่อตรวจสอบหลายเงื่อนไขตามลำดับ:

const temperature = 75

if (temperature > 80) {
  console.log("It's hot outside!")
} else if (temperature > 60) {
  console.log("Perfect weather!") // บรรทัดนี้จะทำงาน
} else if (temperature > 40) {
  console.log("It's a bit chilly")
} else {
  console.log("It's cold!")
}

สำคัญ: มีเพียงเงื่อนไข แรก ที่ตรงเท่านั้นที่จะทำงาน แม้หลายเงื่อนไขจะเป็นจริงก็ตาม

ตัวดำเนินการทางตรรกะ

รวมหลายเงื่อนไขเข้าด้วยกัน:

AND (&&) - ทั้งสองเงื่อนไขต้องเป็นจริง

const age = 25
const hasLicense = true

if (age >= 18 && hasLicense) {
  console.log("You can drive!") // ทั้งสองเงื่อนไขเป็นจริง
}

OR (||) - อย่างน้อยหนึ่งเงื่อนไขต้องเป็นจริง

const isWeekend = false
const isHoliday = true

if (isWeekend || isHoliday) {
  console.log("No work today!") // หนึ่งเงื่อนไขเป็นจริง
}

NOT (!) - กลับค่าเงื่อนไข

const isLoggedIn = false

if (!isLoggedIn) {
  console.log("Please log in") // ทำงานเพราะ !false คือ true
}

ค่า Truthy และ Falsy

JavaScript ถือว่าค่าบางค่าเป็น “truthy” หรือ “falsy” ในคำสั่ง If:

ค่า Falsy (ถือเป็น false):

  • false
  • 0
  • "" (String ว่าง)
  • null
  • undefined
  • NaN

ค่า Truthy (ถือเป็น true):

  • ค่าอื่นทั้งหมด! รวมถึง "0", [], {}
const name = "Kyle"

if (name) {
  console.log(`Hello ${name}!`) // ทำงานเพราะ String ที่ไม่ว่างเป็น truthy
}

const emptyString = ""
if (emptyString) {
  console.log("This won't run") // String ว่างเป็น falsy
}

คำสั่ง If ซ้อนกัน

คุณสามารถใส่คำสั่ง If ไว้ในคำสั่ง If อื่นได้:

const weather = "sunny"
const temperature = 75

if (weather === "sunny") {
  if (temperature > 70) {
    console.log("Perfect day for the beach!") // บรรทัดนี้จะทำงาน
  } else {
    console.log("Sunny but a bit cold")
  }
} else {
  console.log("Not a sunny day")
}

โดยทั่วไปไม่แนะนำให้ทำเช่นนี้หากไม่จำเป็น เพราะอาจทำให้โค้ดอ่านยากขึ้น

Guard Clause

Guard clause เป็นวิธีจัดการเงื่อนไขตั้งแต่ต้นและออกจากฟังก์ชันหรือบล็อกหากเงื่อนไขไม่ผ่าน ช่วยให้ตรรกะหลักซ้อนกันน้อยลงและอ่านง่ายขึ้น:

const weather = "sunny"
const temperature = 75

if (weather !== "sunny") {
  console.log("Not a sunny day")
  return // ออกจากการทำงานก่อน
}

if (temperature > 70) {
  console.log("Perfect day for the beach!") // บรรทัดนี้จะทำงาน
} else {
  console.log("Sunny but a bit cold")
}

คำสั่ง If บรรทัดเดียว

หากมีคำสั่งเดียวในบล็อก If หรือ Else คุณสามารถละวงเล็บปีกกา {} ได้:

const age = 18
if (age >= 18) console.log("You can vote!") // บรรทัดนี้จะทำงาน
else console.log("You cannot vote yet") // บรรทัดนี้จะไม่ทำงาน

โดยทั่วไปผมใช้รูปแบบนี้เฉพาะกับ guard clause เพราะการใช้บ่อยเกินไปทำให้โค้ดอ่านยากขึ้น

แบบฝึกหัด

ลองเขียนคำสั่ง If ที่:

  1. ตรวจสอบว่าบุคคลสามารถขอใบขับขี่ได้หรือไม่
  2. ข้อกำหนด: ต้องมีอายุอย่างน้อย 16 ปี และเรียนหลักสูตรขับรถจบแล้ว
  3. หากขอใบขับขี่ได้ ให้แสดง “You can get your license!”
  4. หากอายุน้อยเกินไป ให้แสดง “You must be at least 16”
  5. หากยังเรียนหลักสูตรขับรถไม่จบ ให้แสดง “You need to complete driver's education”
เฉลย
const age = 17
const hasDriversEd = true

if (age < 16) {
  console.log("You must be at least 16")
} else if (!hasDriversEd) {
  console.log("You need to complete driver's education")
} else {
  console.log("You can get your license!")
}

อีกวิธีหนึ่งโดยใช้ guard clause:

const age = 17
const hasDriversEd = true

if (age < 16) {
  console.log("You must be at least 16")
  return
}

if (!hasDriversEd) {
  console.log("You need to complete driver's education")
  return
}

console.log("You can get your license!")
