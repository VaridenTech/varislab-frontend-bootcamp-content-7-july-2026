การใช้งาน Keyword this และ new

Keyword this และ new เป็นสองแนวคิดที่มักจะสร้างความสับสนให้กับผู้เริ่มต้นเขียน JavaScript มากที่สุด แต่ในขณะเดียวกันก็เป็นฟีเจอร์ที่ทรงพลังอย่างมาก การทำความเข้าใจการทำงานของ this และ new จะช่วยให้คุณสามารถเขียนโปรแกรมในรูปแบบ Object-Oriented Programming (OOP) ทำงานกับ Class และเรียกใช้เครื่องมือสำเร็จรูปต่าง ๆ ใน JavaScript ได้อย่างเชี่ยวชาญ

💡 คำแนะนำ: หัวข้อนี้มีเนื้อหาค่อนข้างเป็นนามธรรมและซับซ้อน แนะนำให้ค่อย ๆ อ่านและลองรันโค้ดตัวอย่างตามเพื่อดูพฤติกรรมของมันนะครับ

1. Keyword this คืออะไร?

Keyword this คือตัวแทนที่ใช้สำหรับ อ้างอิงถึง Object (Object) ตัวใดตัวหนึ่ง แต่ความน่าปวดหัวคือ this จะไม่ได้ชี้ไปยัง Object เดิมเสมอไป โดยค่าของมันจะเปลี่ยนไปตาม "บริบทและวิธีการเรียกใช้งาน" (Execution Context) ณ ขณะนั้น ให้จินตนาการว่า this เป็นตัวแปรพิเศษที่ JavaScript สร้างขึ้นให้เราใช้งานโดยอัตโนมัติ

console.log(this) // ในบริบททั่วไปบนเว็บ Browser (Global Context) this จะอ้างถึง Window Object เสมอ

การใช้งาน this ภายใน Object

เมื่อเราเรียกใช้ this ภายใน Function (Method) ที่อยู่ภายใต้ Object ใด ๆ ตัว this จะอ้างอิงถึง ตัว Object ที่ครอบมันอยู่ ณ ขณะนั้น เสมอ:

const person = {
  name: "Alice",
  age: 25,
  greet() {
    console.log(`Hello, my name is ${this.name}`)
  },
}

person.greet() // "Hello, my name is Alice"

ทำให้เราสามารถเข้าถึงข้อมูลคุณสมบัติ (Properties) อื่น ๆ ภายใน Object เดียวกันได้อย่างสะดวกผ่าน this.propertyName

2. ทำความเข้าใจเรื่อง Class

Class (Class) เปรียบเสมือนพิมพ์เขียวหรือแม่แบบ (Blueprint) ที่เราออกแบบไว้เพื่อใช้สำหรับสร้าง Object ที่มีโครงสร้างเหมือน ๆ กันในปริมาณมาก โดยภายใน Class เราจะใช้ Keyword this เพื่ออ้างอิงถึง Object ตัวจริงที่ถูกสร้างขึ้นมาจาก Class นั้น ๆ (เรียกว่า Instance)

ในการประกาศสร้าง Class ใน JavaScript เราจะใช้ Keyword class ตามด้วยชื่อ Class (นิยมขึ้นต้นด้วยตัวพิมพ์ใหญ่) และเขียนโครงสร้างทั้งหมดไว้ภายในวงเล็บปีกกา {}:

class Person {
  constructor(name, age) {
    this.name = name
    this.age = age
  }

  greet() {
    console.log(`Hello, my name is ${this.name}`)
  }
}

const alice = new Person("Alice", 25)
alice.greet() // "Hello, my name is Alice"

การใช้ Class จะช่วยให้เราสามารถห่อหุ้มข้อมูล (Properties) และพฤติกรรมการทำงาน (Methods) ที่เกี่ยวข้องกันเอาไว้ด้วยกันอย่างเป็นระเบียบและเป็นสัดส่วน

โครงสร้างและส่วนประกอบสำคัญของ Class

จากตัวอย่างโค้ดด้านบน เราสามารถแบ่งส่วนประกอบสำคัญได้ดังนี้:

1. Function สร้าง Object (Constructor)

Method constructor() คือ Function พิเศษที่จะ ทำงานโดยอัตโนมัติทันทีที่มีการสร้าง Object ใหม่ขึ้นมาจาก Class นี้ ทำหน้าที่สำหรับรับค่าเริ่มต้นเข้ามาตั้งค่าให้กับ Object:

class Person {
  constructor(name, age) {
    this.name = name
    this.age = age
  }
}

เบื้องหลังการทำงานของ Constructor เปรียบเสมือนกระบวนการดังนี้:

  1. สร้าง Object ว่างขึ้นมาใหม่: ระบบจะจำลอง this = {} ขึ้นมาโดยอัตโนมัติ
  2. กำหนดคุณสมบัติให้กับ Object: โค้ดที่เราเขียนจะนำค่าที่รับมาไปใส่ไว้ใน this (เช่น this.name = name)
  3. ส่ง Object นั้นกลับออกไป: ระบบจะส่ง this ที่มีข้อมูลครบถ้วนกลับออกไปใช้งานโดยอัตโนมัติ

2. การประกาศ Method (Methods)

เราสามารถเขียน Function การทำงานต่าง ๆ ไว้ภายใน Class ได้โดยตรง โดยใช้โครงสร้างไวยากรณ์แบบย่อ (ไม่ต้องใส่ Keyword function):

class Person {
  greet() {
    console.log(`Hello, my name is ${this.name}`)
  }
}

Method เหล่านี้จะสามารถเข้าถึง Keyword this เพื่อดึงข้อมูลของคุณสมบัติภายใน Object นั้น ๆ มาใช้งานได้โดยอัตโนมัติ

3. การสร้าง Object จริง (Instantiation ด้วย Keyword new)

เมื่อเราต้องการนำพิมพ์เขียว (Class) มาสร้างเป็น Object ตัวจริงเพื่อใช้งาน เราจะต้องใช้ Keyword new นำหน้าชื่อ Class เสมอ:

const alice = new Person("Alice", 25)

คำสั่งนี้จะไปสั่งให้ Function constructor ทำงานเพื่อสร้างและส่ง Object ตัวใหม่ที่มีคุณสมบัติและ Method ตามพิมพ์เขียวกลับมาให้เราใช้งาน

ข้อแนะนำเพิ่มเติมเกี่ยวกับ Class

แม้ว่า Class จะมีฟีเจอร์ขั้นสูงอื่น ๆ อีกมากมาย (เช่น การสืบทอดคุณสมบัติ หรือ Inheritance) แต่สิ่งสำคัญที่สุดสำหรับผู้เริ่มต้นคือการเข้าใจว่า Class คืออะไร และ Keyword this กับ new ทำงานร่วมกันอย่างไร

ในการพัฒนาเว็บยุคปัจจุบัน (เช่น การเขียน React ด้วย Functional Components) นักพัฒนาอาจจะไม่ได้เขียน Class บ่อยเท่าในอดีต แต่การเข้าใจโครงสร้างนี้ยังคงมีความสำคัญอย่างยิ่งเพราะเครื่องมือและไลบรารีส่วนใหญ่ยังคงทำงานอยู่บนพื้นฐานนี้

3. ตัวอย่างการใช้งาน Object สำเร็จรูปใน JavaScript

ในภาษา JavaScript มี Object สำเร็จรูป (Built-in Objects) ที่ถูกสร้างขึ้นจาก Class ระบบอยู่มากมาย ซึ่งเราจำเป็นต้องใช้ Keyword new ในการเรียกใช้งาน เช่น Object จัดการวันที่และเวลา (Date):

// สร้าง Object จัดการวันที่และเวลาปัจจุบัน
const now = new Date()
console.log(now) // แสดงวันที่และเวลาปัจจุบัน

// Object Date จะใช้ this ภายในเพื่อดึงข้อมูลย่อยออกมา
console.log(now.getMonth())
console.log(now.getDay())
console.log(now.getDate())
