คีย์เวิร์ด this/new

คีย์เวิร์ด this/new เป็นสองแนวคิดที่ชวนสับสนที่สุดใน JavaScript แต่ก็ทรงพลังอย่างมาก การเข้าใจ this/new จำเป็นต่อการทำงานกับ Object, Class และคุณสมบัติที่มีมาให้มากมายใน JavaScript

⚠️ คำเตือน: หัวข้อนี้เข้าใจยาก จึงควรใช้เวลาทำความเข้าใจให้ถ่องแท้!

this คืออะไร?

คีย์เวิร์ด this อ้างถึง Object หนึ่ง แต่จะเป็น Object ใด นั้นขึ้นอยู่กับวิธีและตำแหน่งที่ใช้ this ให้คิดว่า this เป็นตัวแปรพิเศษที่ JavaScript สร้างให้อัตโนมัติ และค่าของมันกำหนดจากบริบทที่ใช้งาน

console.log(this) // ในเบราว์เซอร์ this จะอ้างถึง window object โดยค่าเริ่มต้น

this กับ Object

เมื่อใช้ this ภายในเมธอดของ Object มันจะอ้างถึง Object นั้นเอง:

const person = {
  name: "Alice",
  age: 25,
  greet() {
    console.log(`Hello, my name is ${this.name}`)
  },
}

person.greet() // "Hello, my name is Alice"

สิ่งนี้มีประโยชน์ในการเข้าถึงพร็อพเพอร์ตีของ Object จากภายในเมธอดของมัน

Class

Class เป็นอีกวิธีหนึ่งในการกำหนด Object และเมธอดของมัน ภายในเมธอดของ Class this จะอ้างถึง instance ของ Class หากต้องการสร้าง Class ใน JavaScript ให้ใช้คีย์เวิร์ด class ตามด้วยชื่อ Class (เริ่มด้วยตัวพิมพ์ใหญ่) แล้วใส่เมธอดและพร็อพเพอร์ตีทั้งหมดไว้ในวงเล็บปีกกา {}

class Person {
  constructor(name, age) {
    this.name = name
    this.age = age
  }

  greet() {
    console.log(`Hello, my name is ${this.name}`)
  }
}

const alice = new Person("Alice", 25)
alice.greet() // "Hello, my name is Alice"

Class เป็นวิธีสร้างแม่แบบสำหรับ Object และห่อหุ้มข้อมูลกับพฤติกรรมที่เกี่ยวข้องไว้ด้วยกัน

ส่วนประกอบของ Class

มาแยกดูส่วนประกอบสำคัญของโค้ดนี้กัน:

Constructor

เมธอด constructor เป็นเมธอดพิเศษที่ทำงานเมื่อสร้าง instance ใหม่ของ Class

class Person {
  constructor(name, age) {
    this.name = name
    this.age = age
  }
}

วิธีคิดเกี่ยวกับ constructor ที่ดีที่สุดคือมองว่าเป็นฟังก์ชันที่:

  1. สร้าง Object ว่างใหม่: this = {} (อัตโนมัติ)
  2. รันโค้ด constructor ของคุณ: โค้ดของคุณกำหนดพร็อพเพอร์ตีบน this
  3. ส่ง Object กลับ: return this (อัตโนมัติ)

การกำหนดเมธอด

คุณกำหนดเมธอดภายใน Class ด้วยไวยากรณ์เดียวกับเมธอดของ Object ได้:

class Person {
  greet() {
    console.log(`Hello, my name is ${this.name}`)
  }
}

ไวยากรณ์โดยพื้นฐานเหมือนการกำหนดฟังก์ชัน เพียงละคีย์เวิร์ด function เมธอดจะเข้าถึง this ซึ่งอ้างถึง instance ของ Class ได้โดยอัตโนมัติ

การสร้าง Instance

หากต้องการสร้าง instance ของ Class ให้ใช้คีย์เวิร์ด new ตามด้วยชื่อ Class และวงเล็บ:

const alice = new Person("Alice", 25)

คำสั่งนี้รัน constructor และส่ง Object ใหม่ที่มีพร็อพเพอร์ตีและเมธอดตามที่กำหนดไว้ใน Class กลับมา

Class แบบเจาะลึก

Class มีคุณสมบัติขั้นสูงให้ใช้อีกมาก แต่ไม่ได้สำคัญเท่าการเข้าใจว่า Class คืออะไร และ this/new ทำงานร่วมกันอย่างไร

โดยส่วนตัวแล้ว ผมไม่ได้ใช้ Class มากนักในโค้ดของตัวเอง แต่คุณอาจใช้บ่อยกว่านั้นได้ตามรูปแบบการเขียนโค้ดของคุณ

ตัวอย่างที่มีมาให้

JavaScript มีตัวอย่างการใช้ new เพื่อสร้าง Object อยู่มากมาย เช่น Object Date สร้างด้วย new Date():

// สร้าง Date object
const now = new Date()
console.log(now) // วันที่และเวลาปัจจุบัน

// Date ใช้ `this` ภายใน
console.log(now.getMonth())
console.log(now.getDay())
console.log(now.getDate())
