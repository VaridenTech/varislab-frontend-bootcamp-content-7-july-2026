Ternary Operator

Ternary operator เป็นวิธีเขียนคำสั่ง if-else แบบง่ายในรูปแบบย่อ เรียกว่า “ternary” เพราะประกอบด้วย operand สามส่วน

ไวยากรณ์พื้นฐาน

const age = 18

age >= 18 ? "trueValue" : "falseValue"

ให้คิดว่า ? คือ “if” และ : คือ “else” หากเงื่อนไขก่อน ? เป็นจริง จะส่งค่าหลัง ? กลับมา มิฉะนั้นจะส่งค่าหลัง : กลับมา

Ternary ส่งค่ากลับ

คุณสมบัติที่มีประโยชน์อีกอย่างคือ ternary operator ส่งค่ากลับเสมอ จึงเหมาะกับการกำหนดค่า:

// if-else แบบเดิม
let message
if (age >= 18) {
  message = "You can vote"
} else {
  message = "You cannot vote"
}
// Ternary operator
const message = age >= 18 ? "You can vote" : "You cannot vote"

การต่อ Ternary Operator

คุณสามารถต่อ ternary หลายชุดเข้าด้วยกันได้ แต่ต้องระวังเพราะอาจอ่านยาก:

// ใช้ได้ แต่อาจอ่านยาก
const grade = score >= 90 ? "A" : score >= 80 ? "B" : score >= 70 ? "C" : "F"

// if-else ชัดเจนกว่า
let grade
if (score >= 90) {
  grade = "A"
} else if (score >= 80) {
  grade = "B"
} else if (score >= 70) {
  grade = "C"
} else {
  grade = "F"
}

เมื่อใดควรใช้ Ternary Operator

✅ เหมาะสำหรับ:

  • เงื่อนไขง่าย ๆ ที่มีผลลัพธ์สองแบบ
  • การกำหนดค่าตามเงื่อนไข
  • นิพจน์สั้น ๆ ที่อ่านง่าย

❌ หลีกเลี่ยงสำหรับ:

  • เงื่อนไขซับซ้อน
  • หลายคำสั่งในแต่ละทางเลือก
  • Ternary operator ที่ซ้อนกัน

แบบฝึกหัด

แปลงคำสั่ง if-else นี้เป็น ternary operator:

let weather
if (temperature > 75) {
  weather = "hot"
} else {
  weather = "not hot"
}
เฉลย
const weather = temperature > 75 ? "hot" : "not hot"
