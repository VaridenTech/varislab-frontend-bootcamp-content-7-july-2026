การใช้งาน Ternary Operator

Ternary Operator คือรูปแบบการเขียนเงื่อนไข if-else แบบย่อให้เหลือเพียงบรรทัดเดียว ซึ่งช่วยให้โค้ดมีความกระชับและสะอาดมากขึ้น ที่ได้ชื่อว่า "Ternary" (สามส่วน) เพราะตัวดำเนินการนี้ประกอบด้วยส่วนประกอบหลัก 3 ส่วนด้วยกัน

โครงสร้างไวยากรณ์พื้นฐาน (Syntax)

const age = 18

age >= 18 ? "trueValue" : "falseValue"

วิธีจำง่าย ๆ คือ ให้มองว่าเครื่องหมายคำถาม ? ทำหน้าที่แทน if และเครื่องหมายโคลอน : ทำหน้าที่แทน else หากเงื่อนไขที่อยู่หน้า ? เป็นจริง (True) ระบบจะเลือกใช้ค่าที่อยู่หลัง ? แต่ถ้าเป็นเท็จ (False) จะเลือกใช้ค่าที่อยู่หลัง : แทน

จุดเด่น: Ternary Operator จะส่งค่ากลับมาเสมอ

คุณสมบัติที่ยอดเยี่ยมของ Ternary Operator คือ มันจะส่งคืนค่าผลลัพธ์ (Return Value) ออกมาเสมอ ทำให้เราสามารถนำมันไปใช้กำหนดค่าเริ่มต้นให้กับตัวแปรได้โดยตรงในบรรทัดเดียว:

// การเขียนแบบ if-else ปกติ
let message
if (age >= 18) {
  message = "You can vote"
} else {
  message = "You cannot vote"
}
// การเขียนย่อด้วย Ternary Operator
const message = age >= 18 ? "You can vote" : "You cannot vote"

การเขียน Ternary Operator ซ้อนกัน (Chaining)

เราสามารถเขียน Ternary Operator ซ้อนกันหลาย ๆ ชั้นเพื่อเปรียบเทียบหลายเงื่อนไขได้ แต่ต้องระมัดระวังเป็นพิเศษเพราะอาจทำให้โค้ดอ่านยากมาก:

// ⚠️ แม้จะทำงานได้ แต่โครงสร้างนี้อ่านยากและสับสนได้ง่าย
const grade = score >= 90 ? "A" : score >= 80 ? "B" : score >= 70 ? "C" : "F"

// ✅ การเขียนแบบ if-else ปกติจะอ่านง่ายและชัดเจนกว่ามากสำหรับกรณีนี้
let grade
if (score >= 90) {
  grade = "A"
} else if (score >= 80) {
  grade = "B"
} else if (score >= 70) {
  grade = "C"
} else {
  grade = "F"
}

เมื่อใดที่ควรใช้ Ternary Operator?

✅ เหมาะสำหรับกรณี:

  • เงื่อนไขที่เรียบง่ายและมีทางเลือกเพียง 2 ทางชัดเจน
  • การกำหนดค่าให้กับตัวแปรตามเงื่อนไข
  • ชุดคำสั่งสั้น ๆ ที่จบได้ในบรรทัดเดียวและอ่านเข้าใจง่าย

❌ ควรหลีกเลี่ยงในกรณี:

  • เงื่อนไขที่มีความซับซ้อนสูงหรือมีหลายทางเลือก
  • เมื่อมีชุดคำสั่งที่ต้องทำงานหลายบรรทัดในแต่ละเงื่อนไข
  • การเขียน Ternary Operator ซ้อนกันหลาย ๆ ชั้น (Nested Ternary)

ลองทำแบบฝึกหัด

จงแปลงคำสั่งเงื่อนไข if-else ต่อไปนี้ให้เป็นรูปแบบย่อโดยใช้ Ternary Operator:

let weather
if (temperature > 75) {
  weather = "hot"
} else {
  weather = "not hot"
}
ดูเฉลย 
const weather = temperature > 75 ? "hot" : "not hot"
