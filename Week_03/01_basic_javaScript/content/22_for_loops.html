การใช้งาน For Loop

For Loop คือคำสั่งควบคุมการทำงานซ้ำ (Loop) ที่ใช้เมื่อเราทราบจำนวนรอบการทำงานที่แน่นอนล่วงหน้า เหมาะสำหรับการนับจำนวนรอบ การวนลูปดึงข้อมูลใน Array (Array) หรือการทำงานซ้ำ ๆ ที่มีรูปแบบชัดเจน

1. โครงสร้างไวยากรณ์ For Loop พื้นฐาน

คำสั่ง For Loop จะประกอบด้วยส่วนควบคุมหลัก 3 ส่วนที่เขียนอยู่ภายในวงเล็บ () และคั่นด้วยเครื่องหมายอัฒภาค (Semicolon - ;) ตามด้วยบล็อกคำสั่งที่ต้องการให้ทำงานซ้ำในวงเล็บปีกกา {}:

for (let i = 0; i < 5; i++) {
  console.log(`Count: ${i}`)
}
// ผลลัพธ์: Count: 0, 1, 2, 3, 4
  1. let i = 0 (Initialization): ส่วนกำหนดค่าเริ่มต้นของตัวแปรควบคุมลูป จะทำงานเพียง ครั้งเดียว ตอนเริ่มต้นลูป
  2. i < 5 (Condition): เงื่อนไขที่ระบบจะตรวจสอบ ก่อนเริ่มทำงานในแต่ละรอบ หากเงื่อนไขเป็นจริง (True) จะทำงานในลูปต่อ แต่หากเป็นเท็จ (False) จะหยุดการทำงานและออกจากลูปทันที
  3. i++ (Update): ส่วนอัปเดตค่าของตัวแปรควบคุมลูป จะทำงาน หลังจากที่ทำคำสั่งในรอบนั้น ๆ เสร็จสิ้นแล้ว (ในที่นี้คือการเพิ่มค่าขึ้นทีละ 1)
  4. { ... } (Body): บล็อกของชุดคำสั่งที่จะทำงานซ้ำในแต่ละรอบ

ตัวอย่างเปรียบเทียบการทำงานซ้ำ

สมมติว่าเราต้องการแสดงคำว่า "Hi" ออกทาง Console ทั้งหมด 5 ครั้ง:

// ❌ แบบไม่ใช้ Loop (โค้ดซ้ำซ้อนและแก้ไขยาก)
console.log("Hi")
console.log("Hi")
console.log("Hi")
console.log("Hi")
console.log("Hi")

// ✅ แบบใช้ For Loop (สะอาด กระชับ และดูแลรักษาง่ายกว่า)
for (let i = 0; i < 5; i++) {
  console.log("Hi")
}

โค้ดทั้งสองแบบให้ผลลัพธ์เหมือนกันทุกประการ แต่การใช้ For Loop จะช่วยให้เราปรับเปลี่ยนจำนวนรอบการทำงานได้ง่ายมาก (เช่น เปลี่ยนจาก 5 รอบเป็น 100 รอบได้ในจุดเดียว)

2. รูปแบบการเขียน For Loop ที่พบบ่อย

  • การนับขึ้น (Counting Up):

    // นับเลขจาก 1 ถึง 10
for (let i = 1; i <= 10; i++) console.log(i)

  • การนับลง (Counting Down):

    // นับถอยหลังจาก 10 ลงมาถึง 1
for (let i = 10; i >= 1; i--) console.log(i)

  • การข้ามตัวเลข (Stepping):

    // แสดงเฉพาะเลขคู่ (เพิ่มค่าทีละ 2)
for (let i = 0; i <= 10; i += 2) console.log(i) // 0, 2, 4, 6, 8, 10

// แสดงตัวเลขเพิ่มขึ้นทีละ 3
for (let i = 0; i <= 20; i += 3) console.log(i) // 0, 3, 6, 9, 12, 15, 18

3. การวนลูปดึงข้อมูลจาก Array (Iterating Arrays)

การใช้ For Loop ร่วมกับคุณสมบัติ .length ของ Array เป็นวิธี Class สิกในการดึงข้อมูลทุกตัวใน Array ออกมาใช้งาน:

const fruits = ["apple", "banana", "orange", "grape"]

for (let i = 0; i < fruits.length; i++) {
  console.log(`${i}: ${fruits[i]}`)
}
// ผลลัพธ์: 0: apple, 1: banana, 2: orange, 3: grape

4. การเขียน For Loop ซ้อนกัน (Nested Loops)

เราสามารถเขียน For Loop ซ้อนอยู่ภายใน For Loop อีกตัวหนึ่งได้ เหมาะสำหรับโครงสร้างข้อมูลแบบหลายมิติ หรือการทำงานที่เป็นตาราง:

// ตัวอย่างการสร้างตารางสูตรคูณแม่ 1 ถึง 3
for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(`${i} × ${j} = ${i * j}`)
  }
}

5. คำสั่งควบคุมการทำงานของ Loop (Loop Control Statements)

  • คำสั่ง break (หยุดการทำงานของลูปทันที)

    const numbers = [1, 3, 7, 2, 9, 4]
for (let i = 0; i < numbers.length; i++) {
  if (numbers[i] > 5) {
    console.log(`Found: ${numbers[i]} at index ${i}`)
    break // สั่งให้ออกจากลูปทันทีเมื่อเจอค่าที่ต้องการ
  }
}

  • คำสั่ง continue (ข้ามรอบปัจจุบันไปรอบถัดไป)

    for (let i = 1; i <= 5; i++) {
  if (i === 3) continue // ข้ามรอบที่ i เป็น 3 ไปรอบถัดไปทันที
  console.log(i)
}
// ผลลัพธ์: 1, 2, 4, 5

6. ทางเลือกอื่นในการวนลูป (Modern Alternatives)

  • คำสั่ง for...of (สำหรับดึงข้อมูลใน Array โดยตรง)

    const fruits = ["apple", "banana", "orange"]

for (let i = 0; i < fruits.length; i++) console.log(fruits[i])

for (const fruit of fruits) console.log(fruit)

  • คำสั่ง for...in (สำหรับดึงคีย์ต่าง ๆ ใน Object)

    const person = { name: "Kyle", age: 30, city: "New York" }
for (const key in person) {
  console.log(`${key}: ${person[key]}`)
}

7. ข้อผิดพลาดที่พบบ่อย (Common Pitfalls)

  1. เกิดลูปไม่สิ้นสุด (Infinite Loop)

    // ❌ Loop ไม่สิ้นสุด
for (let i = 0; i < 10; i--) console.log(i)

    ⚠️ ข้อควรระวัง: หากเงื่อนไขของลูปไม่มีวันเป็นเท็จ ลูปจะทำงานไปเรื่อย ๆ ไม่มีวันสิ้นสุด ส่งผลให้เว็บ Browser ค้าง เครื่องทำงานช้าลง หรือโปรแกรมแครชได้

  2. ข้อผิดพลาดคลาดเคลื่อนหนึ่งตำแหน่ง (Off-by-One Error)

    const arr = [10, 20, 30]
for (let i = 0; i < arr.length - 1; i++) console.log(arr[i]) // ❌ ผลลัพธ์ขาดตัวสุดท้าย (เพราะใช้ i < arr.length - 1)
for (let i = 0; i <= arr.length; i++) console.log(arr[i]) // ❌ ดึงข้อมูลเกินขอบเขตของ Array (รอบสุดท้ายจะเป็น undefined)
for (let i = 0; i < arr.length; i++) console.log(arr[i]) // ✅ วิธีการเขียนที่ถูกต้องและปลอดภัยที่สุด

  3. การแก้ไขค่าตัวแปรควบคุมลูปภายในบล็อกคำสั่ง

    // ❌ ไม่แนะนำ: การปรับเปลี่ยนค่า i ภายในบล็อกลูปจะทำให้ตรวจหา Bug ยาก
for (let i = 0; i < 5; i++) {
  console.log(i)
  if (i === 1) i += 2
}

// ✅ วิธีที่ถูกต้อง: ปล่อยให้การอัปเดตค่า i เป็นหน้าที่ของส่วน Update ในวงเล็บด้านบน
for (let i = 0; i < 5; i++) console.log(i)

ลองทำแบบฝึกหัด

จงเขียน For Loop เพื่อทำงานดังต่อไปนี้:
1. สร้าง Array เก็บตัวเลขคู่ 10 ตัวแรก (2, 4, 6, ..., 20)
2. เขียน For Loop ตัวที่สองเพื่อหาผลรวมของตัวเลขทั้งหมดใน Array นั้น
3. แสดงผลลัพธ์ Array และผลรวมออกทาง Console

ดูเฉลย 
const evenNumbers = []
for (let i = 1; i <= 10; i++) evenNumbers.push(i * 2)

let sum = 0
for (let i = 0; i < evenNumbers.length; i++) sum += evenNumbers[i]

console.log("Even numbers:", evenNumbers)
console.log("Sum:", sum) // 110
