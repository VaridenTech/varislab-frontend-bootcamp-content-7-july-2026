For Loop

For loop ใช้เมื่อต้องทำโค้ดซ้ำตามจำนวนครั้งที่กำหนด เหมาะสำหรับการนับ การวนผ่าน Array หรืองานที่ต้องทำซ้ำ

ไวยากรณ์ For Loop พื้นฐาน

For loop มีสามส่วนหลักคั่นด้วยอัฒภาค และมีส่วนคำสั่งอยู่ในวงเล็บปีกกา:

for (let i = 0; i < 5; i++) {
  console.log(`Count: ${i}`)
}
// ผลลัพธ์: Count: 0, 1, 2, 3, 4
  1. let i = 0 - ทำงานครั้งเดียวตอนเริ่ม มักสร้างตัวแปรติดตามจำนวนรอบ
  2. i < 5 - เงื่อนไขที่ตรวจก่อนแต่ละรอบ หากจริงจะรันโค้ด หากเท็จจะหยุด
  3. i++ - ทำงานหลังแต่ละรอบ โดยปกติใช้อัปเดตตัวแปร Loop
  4. { ... } - โค้ดที่ทำงานในแต่ละรอบ

ตัวอย่างการนับอย่างง่าย

แสดง "Hi" ห้าครั้ง:

// ไม่มี Loop (ซ้ำซ้อน)
console.log("Hi")
console.log("Hi")
console.log("Hi")
console.log("Hi")
console.log("Hi")

// ใช้ For loop (สะอาดกว่า)
for (let i = 0; i < 5; i++) {
  console.log("Hi")
}

ทั้งสองให้ผลเหมือนกัน แต่ For loop ดูแลรักษาง่ายกว่ามาก

รูปแบบ Loop ที่พบบ่อย

  • นับขึ้น

    // นับจาก 1 ถึง 10
for (let i = 1; i <= 10; i++) console.log(i)

  • นับลง

    // นับถอยหลังจาก 10 ถึง 1
for (let i = 10; i >= 1; i--) console.log(i)

  • ข้ามตัวเลข

    // เฉพาะเลขคู่
for (let i = 0; i <= 10; i += 2) console.log(i) // 0, 2, 4, 6, 8, 10

// ทุก ๆ สามตัวเลข
for (let i = 0; i <= 20; i += 3) console.log(i) // 0, 3, 6, 9, 12, 15, 18

การวนผ่าน Array

For loop เหมาะสำหรับการวนผ่านสมาชิกทุกตัวใน Array:

const fruits = ["apple", "banana", "orange", "grape"]

for (let i = 0; i < fruits.length; i++) {
  console.log(`${i}: ${fruits[i]}`)
}
// ผลลัพธ์: 0: apple, 1: banana, 2: orange, 3: grape

For Loop ซ้อนกัน

คุณใส่ For loop ไว้ภายใน For loop อื่นได้:

// ตารางสูตรคูณ
for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(`${i} × ${j} = ${i * j}`)
  }
}

คำสั่งควบคุม Loop

  • break - ออกจาก Loop ก่อนกำหนด

    const numbers = [1, 3, 7, 2, 9, 4]
for (let i = 0; i < numbers.length; i++) {
  if (numbers[i] > 5) {
    console.log(`Found: ${numbers[i]} at index ${i}`)
    break // ออกจาก Loop ทันที
  }
}

  • continue - ข้ามไปยังรอบถัดไป

    for (let i = 1; i <= 5; i++) {
  if (i === 3) continue // ข้าม 3
  console.log(i)
}
// ผลลัพธ์: 1, 2, 4, 5

ทางเลือกแทน For Loop

  • For...of (สำหรับ Array)

    const fruits = ["apple", "banana", "orange"]

for (let i = 0; i < fruits.length; i++) console.log(fruits[i])

for (const fruit of fruits) console.log(fruit)

  • For...in (สำหรับพร็อพเพอร์ตีของ Object)

    const person = { name: "Kyle", age: 30, city: "New York" }
for (const key in person) {
  console.log(`${key}: ${person[key]}`)
}

ข้อผิดพลาดที่พบบ่อย

  1. Loop ไม่สิ้นสุด: ต้องแน่ใจว่าเงื่อนไขจะกลายเป็นเท็จในที่สุด

    // ❌ Loop ไม่สิ้นสุด
for (let i = 0; i < 10; i--) console.log(i)

    สัญญาณคือเบราว์เซอร์ไม่ตอบสนอง Console แสดงสิ่งเดิมไม่หยุด หรือต้องบังคับปิดโปรแกรม

  2. ข้อผิดพลาดคลาดหนึ่งตำแหน่ง

    const arr = [10, 20, 30]
for (let i = 0; i < arr.length - 1; i++) console.log(arr[i]) // ❌ ขาดตัวสุดท้าย
for (let i = 0; i <= arr.length; i++) console.log(arr[i]) // ❌ เกินขอบเขต
for (let i = 0; i < arr.length; i++) console.log(arr[i]) // ✅ ถูกต้อง

  3. แก้ไขตัวแปร Loop ภายใน Loop

    // ❌ เปลี่ยนตัวแปร Loop
for (let i = 0; i < 5; i++) {
  console.log(i)
  if (i === 1) i += 2
}

// ✅ อย่าเปลี่ยนตัวแปร Loop
for (let i = 0; i < 5; i++) console.log(i)

แบบฝึกหัด

เขียน For loop ที่สร้าง Array ของเลขคู่ 10 ตัวแรก จากนั้นคำนวณผลรวมใน Loop ที่สอง และแสดง Array กับผลรวม

เฉลย
const evenNumbers = []
for (let i = 1; i <= 10; i++) evenNumbers.push(i * 2)

let sum = 0
for (let i = 0; i < evenNumbers.length; i++) sum += evenNumbers[i]

console.log("Even numbers:", evenNumbers)
console.log("Sum:", sum) // 110
