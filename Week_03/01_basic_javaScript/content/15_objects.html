การใช้งาน Object

Object เป็นประเภทข้อมูลที่สำคัญและถูกใช้งานบ่อยที่สุดในภาษา JavaScript เนื่องจากช่วยให้เราสามารถจัดระเบียบและรวมกลุ่มข้อมูลที่เกี่ยวข้องกันเอาไว้ด้วยกันได้อย่างเป็นระบบ

Object คืออะไร?

Object คือโครงสร้างข้อมูลที่เก็บข้อมูลในรูปแบบของคู่ คีย์-ค่า (Key-Value Pairs) เปรียบเสมือนกล่องเก็บของขนาดใหญ่ที่มีป้ายกำกับ (Key) ติดไว้ที่แต่ละช่อง เพื่อระบุว่าช่องนั้นเก็บข้อมูลอะไร (Value) เอาไว้

// การประกาศตัวแปรแยกกัน (กระจัดกระจายและยากต่อการจัดการ)
const name = "Kyle"
const age = 30
const favoriteNumber = 3

// การใช้ Object เพื่อจัดรวมกลุ่มข้อมูลให้อยู่ในที่เดียวกัน
const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
}

วิธีการสร้าง Object

เราจะใช้สัญลักษณ์วงเล็บปีกกา {} (Curly Braces) ในการประกาศสร้าง Object และใช้เครื่องหมายจุลภาค (,) คั่นระหว่างคู่ Key-Value แต่ละตัว:

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
}

โครงสร้างคู่ Key-Value

ข้อมูลทุกตัวภายใน Object จะต้องประกอบด้วย 2 ส่วนเสมอ คือ:

  • Key (คีย์): เปรียบเสมือนชื่อป้ายกำกับหรือชื่อคุณสมบัติ (Property Name)
  • Value (ค่า): ข้อมูลจริง ๆ ที่เราต้องการจัดเก็บ ซึ่งสามารถเป็นข้อมูลประเภทใดก็ได้

ตามหลักปฏิบัติที่ดี ตัว Key ควรตั้งชื่อโดยใช้รูปแบบ camelCase และไม่มีช่องว่าง ส่วน Value สามารถเก็บข้อมูลประเภทใดก็ได้ ไม่ว่าจะเป็นตัวเลข ข้อความ Boolean หรือแม้กระทั่ง Array และ Object อื่น ๆ โดยเราจะใช้เครื่องหมายโคลอน (:) คั่นระหว่าง Key และ Value เสมอ

const person = {
  name: "Kyle", // key คือ name, value คือ "Kyle"
  age: 30, // key คือ age, value คือ 30
  favoriteNumber: 3, // key คือ favoriteNumber, value คือ 3
}

การเข้าถึงข้อมูลใน Object (Accessing Properties)

วิธีการเข้าถึงข้อมูลใน Object ที่นิยมและง่ายที่สุดคือการใช้ Dot Notation (ใช้เครื่องหมายจุด . ตามด้วยชื่อ Key):

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
}

console.log(person.name) // "Kyle"
console.log(person.age) // 30
console.log(person.favoriteNumber) // 3

การเขียน Function ภายใน Object (Methods)

Object ไม่เพียงแต่เก็บข้อมูลทั่วไปได้เท่านั้น แต่ยังสามารถเก็บ Function การทำงานไว้ข้างในได้ด้วย ซึ่งเราจะเรียก Function ที่อยู่ภายใน Object ว่า Method (Methods):

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
  sayHi: function () {
    console.log("Hi")
  },
}

person.sayHi() // "Hi"

การเขียน Method แบบย่อ (Shorthand Method Syntax)

ใน JavaScript เวอร์ชันปัจจุบัน (ES6+) เราสามารถเขียน Method ภายใน Object ได้สั้นลงโดยไม่จำเป็นต้องใส่ Keyword function และเครื่องหมายโคลอน ::

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
  sayHi() {
    console.log("Hi")
  },
  sayGoodbye() {
    console.log("Goodbye")
  },
}

person.sayHi() // "Hi"
person.sayGoodbye() // "Goodbye"

ตัวอย่างการใช้งานจริง: คำสั่ง console.log

จริง ๆ แล้วคุณได้ใช้งาน Object มาตั้งแต่วันแรกที่เขียน JavaScript! เพราะคำสั่งยอดนิยมอย่าง console.log() นั้น ตัว console ก็คือ Object ตัวหนึ่ง และ log ก็คือ Method (Function) ที่อยู่ข้างในนั่นเอง:

console.log("Hello") // console คือ Object และ log คือ Method (Method) ที่อยู่ข้างใน

// คล้ายกับ:
const console = {
  log(message) {
    // คำสั่งสำหรับแสดงข้อความออกทางหน้าจอ
  },
}

การเข้าถึงข้อมูลด้วย Bracket Notation

นอกจากเครื่องหมายจุด . แล้ว เรายังสามารถเข้าถึงข้อมูลใน Object ได้โดยใช้สัญลักษณ์วงเล็บเหลี่ยม [] และระบุชื่อ Key เป็นข้อความ (String) ได้เช่นกัน:

const car = {
  make: "Nissan",
  model: "370Z",
}

console.log(car["make"]) // "Nissan"
console.log(car.make) // "Nissan" (แนะนำให้ใช้แบบนี้เป็นหลัก)

กรณีหลักที่เราจำเป็นต้องใช้ Bracket Notation คือเมื่อชื่อ Key ที่เราต้องการเข้าถึงนั้นถูกจัดเก็บอยู่ภายในตัวแปรอีกทีหนึ่ง (ทำให้ชื่อ Key มีความยืดหยุ่นตามตัวแปร):

const property = "make"
console.log(car[property]) // "Nissan"

Object ซ้อน Object (Nested Objects)

Object สามารถเก็บข้อมูลที่มีความซับซ้อนสูงได้ โดยการนำ Object อื่นหรือ Array มาซ้อนไว้ข้างในเป็น Value:

const person = {
  name: "Kyle",
  address: {
    street: "12345 Main Street",
    city: "Somewhere",
  },
  hobbies: ["weightlifting", "programming"],
}

console.log(person.name) // "Kyle"
console.log(person.address.street) // "12345 Main Street"
console.log(person.address.city) // "Somewhere"
console.log(person.hobbies[0]) // "weightlifting"

ลองทำแบบฝึกหัด

จงสร้าง Object ชื่อ book โดยให้มีคุณสมบัติ (Properties) และ Method ดังต่อไปนี้:

  1. title: เก็บชื่อหนังสือ (String)
  2. author: เก็บข้อมูลผู้แต่งในรูปแบบ Object ที่มี Key ย่อยเป็น firstName และ lastName
  3. yearPublished: เก็บปีที่ตีพิมพ์หนังสือ (Number)
  4. publish: Method (Function) ที่เมื่อเรียกใช้จะแสดงข้อความ "Publishing your book"
ดูเฉลย 
const book = {
  title: "The Way of Kings",
  author: {
    firstName: "Brandon",
    lastName: "Sanderson",
  },
  yearPublished: 2010,
  publish() {
    console.log("Publishing your book")
  },
}
