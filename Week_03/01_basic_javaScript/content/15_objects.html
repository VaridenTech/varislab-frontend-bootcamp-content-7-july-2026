Object

Object เป็นชนิดตัวแปรที่สำคัญที่สุดใน JavaScript เพราะช่วยให้คุณจัดระเบียบข้อมูลได้

Object คืออะไร?

Object คือกลุ่มข้อมูลที่เกี่ยวข้องกัน ซึ่งเก็บรวมกันเป็นคู่ key-value ลองนึกว่าเป็นกล่องที่เก็บข้อมูลหลายส่วนเกี่ยวกับบางสิ่งไว้ด้วยกัน

// ตัวแปรแยกกัน (ไม่ได้จัดระเบียบ)
const name = "Kyle"
const age = 30
const favoriteNumber = 3

// Object (จัดระเบียบไว้ด้วยกัน)
const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
}

การสร้าง Object

ใช้วงเล็บปีกกา {} เพื่อสร้าง Object โดยใช้เครื่องหมายจุลภาคคั่นคู่ key-value

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
}

คู่ Key-Value

ข้อมูลแต่ละส่วนใน Object ประกอบด้วย:

  • Key: ชื่อของพร็อพเพอร์ตี
  • Value: ข้อมูลที่เก็บไว้ในพร็อพเพอร์ตีนั้น

Key ใช้หลักการตั้งชื่อเดียวกับตัวแปร (camelCase และไม่มีช่องว่าง) ส่วน Value เป็นข้อมูลชนิดใดก็ได้ รวมถึง Object หรือฟังก์ชันอื่น โดยใช้เครื่องหมายทวิภาค : คั่น Key กับ Value

const person = {
  name: "Kyle", // key: "name", value: "Kyle"
  age: 30, // key: "age", value: 30
  favoriteNumber: 3, // key: "favoriteNumber", value: 3
}

การเข้าถึงพร็อพเพอร์ตีของ Object

ใช้ dot notation เพื่อเข้าถึงพร็อพเพอร์ตี:

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
}

console.log(person.name) // "Kyle"
console.log(person.age) // 30
console.log(person.favoriteNumber) // 3

การเพิ่มฟังก์ชันลงใน Object

Object สามารถมีฟังก์ชันได้ (มักเรียกว่า เมธอด เมื่ออยู่ภายใน Object):

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
  sayHi: function () {
    console.log("Hi")
  },
}

person.sayHi() // "Hi"

ไวยากรณ์ฟังก์ชันแบบย่อ

คุณสามารถเขียนฟังก์ชันใน Object โดยไม่ใช้คีย์เวิร์ด function:

const person = {
  name: "Kyle",
  age: 30,
  favoriteNumber: 3,
  sayHi() {
    console.log("Hi")
  },
  sayGoodbye() {
    console.log("Goodbye")
  },
}

person.sayHi() // "Hi"
person.sayGoodbye() // "Goodbye"

ตัวอย่างในโลกจริง: console.log

คุณใช้ Object มาตลอด! console คือ Object ที่มีฟังก์ชัน log:

console.log("Hello") // console คือ Object และ log คือเมธอด

// คล้ายกับ:
const console = {
  log(message) {
    // แสดงข้อความ
  },
}

Bracket Notation (อีกวิธีในการเข้าถึง)

คุณสามารถเข้าถึงพร็อพเพอร์ตีด้วยวงเล็บเหลี่ยมและ String ได้เช่นกัน:

const car = {
  make: "Nissan",
  model: "370Z",
}

console.log(car["make"]) // "Nissan"
console.log(car.make) // "Nissan" (แนะนำให้ใช้)

กรณีเดียวที่จำเป็นต้องใช้ bracket notation คือเมื่อชื่อพร็อพเพอร์ตีถูกเก็บไว้ในตัวแปร:

const property = "make"
console.log(car[property]) // "Nissan"

Object ซ้อนกัน

Object สามารถมี Object หรือ Array อื่นอยู่ภายในได้:

const person = {
  name: "Kyle",
  address: {
    street: "12345 Main Street",
    city: "Somewhere",
  },
  hobbies: ["weightlifting", "programming"],
}

console.log(person.name) // "Kyle"
console.log(person.address.street) // "12345 Main Street"
console.log(person.address.city) // "Somewhere"
console.log(person.hobbies[0]) // "weightlifting"

แบบฝึกหัด

สร้าง Object ชื่อ book ที่มีพร็อพเพอร์ตีต่อไปนี้:

  1. title - ชื่อหนังสือ
  2. author - Object ที่มี firstName และ lastName
  3. yearPublished - ปีที่ตีพิมพ์หนังสือ
  4. publish - ฟังก์ชันที่แสดงข้อความ "Publishing your book"
เฉลย
const book = {
  title: "The Way of Kings",
  author: {
    firstName: "Brandon",
    lastName: "Sanderson",
  },
  yearPublished: 2010,
  publish() {
    console.log("Publishing your book")
  },
}
