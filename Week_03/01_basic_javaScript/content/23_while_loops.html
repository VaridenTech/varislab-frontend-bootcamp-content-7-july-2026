While Loop

While loop ใช้เมื่อต้องทำโค้ดซ้ำ แต่ไม่ทราบแน่ชัดว่าต้องวนกี่รอบ โดยจะทำงานต่อไปตราบใดที่เงื่อนไขยังเป็นจริง

ไวยากรณ์พื้นฐานของ While Loop

While loop เริ่มด้วยคีย์เวิร์ด while ตามด้วยเงื่อนไขในวงเล็บ () และบล็อกโค้ดในวงเล็บปีกกา {}:

let i = 0
while (i < 5) {
  // โค้ดที่ต้องทำซ้ำ
  console.log(`Count: ${i}`)
  i++ // สำคัญ: อย่าลืมเปลี่ยนค่า i!
}

Loop จะทำงานต่อไปตราบใดที่เงื่อนไขในวงเล็บเป็น true และหยุดเมื่อกลายเป็น false

การแปลง For Loop เป็น While Loop

นี่คือวิธีแปลง for loop เป็น while loop:

// For loop
for (let i = 0; i < 5; i++) {
  console.log(i)
}

// While loop ที่ทำงานเทียบเท่ากัน
let i = 0 // การกำหนดค่าเริ่มต้น (อยู่นอก Loop)
while (i < 5) { // เงื่อนไข (เหมือนกับ For loop)
  console.log(i) // โค้ดที่ต้องทำงาน
  i++ // การเพิ่มค่า (อยู่ภายใน Loop)
}

ทั้งสองแบบแสดงผล: 0, 1, 2, 3, 4

เมื่อใดควรใช้ While Loop

  1. ไม่ทราบจำนวนรอบ

    // ถามข้อมูลต่อไปจนกว่าผู้ใช้จะป้อน "quit"
let userInput = ""
while (userInput !== "quit") {
  userInput = prompt("Enter a command (or 'quit' to exit):")
  if (userInput !== "quit") {
    console.log(`You entered: ${userInput}`)
  }
}

  2. ประมวลผลโครงสร้างข้อมูลที่ซ้อนกัน

    const person = {
  name: "Kyle",
  friend: {
    name: "Joe",
    friend: {
      name: "Sally",
      friend: null, // Sally ไม่มีเพื่อน
    },
  },
}

// หาเพื่อนคนสุดท้ายในสาย
let currentPerson = person
while (currentPerson.friend !== null) {
  currentPerson = currentPerson.friend
  console.log(`Current friend: ${currentPerson.name}`)
}

console.log(`Final friend: ${currentPerson.name}`) // "Final friend: Sally"

    คุณทำแบบนี้ด้วย for loop ได้ไม่ง่าย เพราะไม่รู้ว่าสายเพื่อนซ้อนลึกเพียงใด!

While Loop กับ For Loop

ใช้ for loop เมื่อ:

  • รู้จำนวนรอบที่แน่นอน
  • กำลังนับหรือวนผ่าน Array
  • มีจุดเริ่ม จุดสิ้นสุด และการเพิ่มค่าที่ชัดเจน

ใช้ while loop เมื่อ:

  • ไม่ทราบว่าต้องวนกี่รอบ
  • ต้องเข้าถึงข้อมูลที่ซ้อนหรือเชื่อมโยงกัน

การควบคุม Loop: break และ continue

คุณใช้ break และ continue ใน while loop ได้เช่นเดียวกับ for loop

Do-While Loop

เป็นรูปแบบที่รันบล็อกโค้ดอย่างน้อยหนึ่งครั้งก่อนตรวจสอบเงื่อนไข:

let userInput
do {
  userInput = prompt("Enter 'yes' to continue:")
  console.log(`You entered: ${userInput}`)
} while (userInput !== "yes")

ความแตกต่างสำคัญคือ do-while ตรวจสอบเงื่อนไข หลังจาก รันบล็อกโค้ด
