การใช้งาน Method จัดการ Array (Array Methods)

Array ในภาษา JavaScript เป็นโครงสร้างข้อมูลที่มีความสามารถสูงมาก และจะยิ่งทรงพลังมากขึ้นเมื่อเราเข้าใจการใช้งาน Method จัดการ Array (Array Methods) ที่ช่วยให้เราสามารถจัดการ ค้นหา แปลงค่า หรือคำนวณข้อมูลใน Array ได้อย่างมีประสิทธิภาพโดยไม่ต้องเขียนลูปแบบเดิม ๆ

1. Method จัดการ Array (Array Methods) คืออะไร?

คือ Function สำเร็จรูปที่ติดมากับตัวแปรประเภท Array ทุกตัวใน JavaScript เราสามารถเรียกใช้งานได้ทันทีผ่านเครื่องหมายจุด (Dot Notation .) และส่ง Function คอลแบ็ก (Callback Function) เข้าไปเพื่อกำหนดการทำงานกับข้อมูลแต่ละตัว:

const numbers = [1, 2, 3, 4, 5]
numbers.forEach((n) => {
  // ทำงานบางอย่างกับตัวเลขแต่ละตัวใน Array
})

2. Method forEach() (การวนลูปแบบกระชับ)

Method forEach() เป็นวิธีที่ง่ายและนิยมที่สุดในการวนลูปเข้าถึงข้อมูลแต่ละตัวใน Array โดยระบบจะนำข้อมูลแต่ละตัวส่งเข้าไปรันใน Function คอลแบ็กที่เรากำหนดไว้ตามลำดับ:

const numbers = [1, 2, 3, 4, 5]
numbers.forEach((number) => console.log(number))
// แสดงผลลัพธ์: 1, 2, 3, 4, 5

การดึงค่าตำแหน่ง (Index) ด้วย forEach()

นอกจากค่าข้อมูลแล้ว Function คอลแบ็กของ forEach() ยังสามารถรับ Parameter ตัวที่สองเป็นตำแหน่งลำดับ (Index) ของข้อมูลแต่ละตัวใน Array ได้อีกด้วย:

const names = ["Kyle", "Sarah", "John"]
names.forEach((name, index) => {
  console.log(`${name} ${index}`)
})
// Kyle 0, Sarah 1, John 2

ซึ่งค่า Index เหล่านี้จะตรงกับตำแหน่งที่เราใช้เข้าถึงข้อมูลผ่านวงเล็บเหลี่ยม [] (Bracket Notation) นั่นเอง:

console.log(numbers[0]) // 1
console.log(numbers[1]) // 2

3. Method map() (การแปลงข้อมูลใน Array)

Method map() ทำหน้าที่เหมือนกับ forEach() ในการวนลูปเข้าถึงข้อมูล แต่มีความแตกต่างสำคัญคือ map() จะสร้างและส่งคืน Array ชุดใหม่ (New Array) กลับมาเสมอ โดยค่าใน Array ใหม่จะเป็นผลลัพธ์ที่ได้จากการแปลงค่าผ่าน Function คอลแบ็ก:

const numbers = [1, 2, 3, 4, 5]
const doubled = numbers.map((number) => {
  return number * 2
})
console.log(doubled) // [2, 4, 6, 8, 10]
console.log(numbers) // Array ต้นฉบับยังคงมีค่าเดิมไม่เปลี่ยนแปลง

คุณสมบัติสำคัญของ map():

  • สร้าง Array ชุดใหม่เสมอ - จะไม่เข้าไปแก้ไขหรือส่งผลกระทบต่อข้อมูลใน Array ต้นฉบับ
  • ขนาดความยาวเท่าเดิม - Array ชุดใหม่ที่ได้จะมีจำนวนสมาชิกเท่ากับ Array ต้นฉบับทุกประการ
  • แปลงโครงสร้างข้อมูลได้ยืดหยุ่น - สามารถนำไปแปลงข้อมูลจากรูปแบบหนึ่งไปเป็นอีกรูปแบบหนึ่งได้ง่าย

ตัวอย่างการนำไปใช้งานจริง (แปลงข้อมูลใน Object)

const people = [
  { id: 1, name: "Alice" },
  { id: 2, name: "Bob" },
  { id: 3, name: "Charlie" },
]
const names = people.map((person) => person.name)
console.log(names) // ["Alice", "Bob", "Charlie"]

4. Method filter() (การคัดกรองข้อมูล)

Method filter() ใช้สำหรับ คัดกรองข้อมูลใน Array โดยจะสร้าง Array ชุดใหม่ที่มีเฉพาะสมาชิกที่ผ่านเงื่อนไขที่เรากำหนดไว้ใน Function คอลแบ็ก (หาก Function ส่งคืนค่าเป็น true จะเก็บข้อมูลนั้นไว้ แต่ถ้าเป็น false จะคัดออก):

const numbers = [1, 2, 3, 4, 5]
const smallNumbers = numbers.filter((number) => number <= 2)
console.log(smallNumbers) // [1, 2]
console.log(numbers) // Array ต้นฉบับยังคงมีค่าเดิมไม่เปลี่ยนแปลง

5. Method find() (การค้นหาข้อมูลตัวแรก)

Method find() ใช้สำหรับค้นหาข้อมูลใน Array โดยจะ ส่งคืนค่าข้อมูลตัวแรกสุดที่ตรงตามเงื่อนไขกลับมาเพียงค่าเดียว และจะหยุดค้นหาทันทีที่เจอ (หากไม่พบข้อมูลใด ๆ ที่ตรงเงื่อนไขเลยจะส่งคืนค่า undefined):

const numbers = [1, 2, 3, 4, 5]
const firstBigNumber = numbers.find((number) => number > 2)
console.log(firstBigNumber) // 3

6. Method some() (ตรวจสอบว่ามีบางตัวตรงเงื่อนไขหรือไม่)

Method some() ใช้สำหรับตรวจสอบว่า มีข้อมูลอย่างน้อย 1 ตัวใน Array ที่ตรงตามเงื่อนไขหรือไม่ โดยจะส่งคืนค่ากลับมาเป็น Boolean (true หรือ false):

const numbers = [1, 2, 3, 4, 5]
const hasLargeNumber = numbers.some((number) => number > 3) // true
const hasNegativeNumber = numbers.some((number) => number < 0) // false

กลไกการทำงานของ some():

  • จะส่งคืนค่า true ทันทีหากพบข้อมูลแม้เพียงตัวเดียวที่ตรงตามเงื่อนไข
  • จะส่งคืนค่า false ก็ต่อเมื่อไม่มีข้อมูลตัวใดเลยที่ตรงตามเงื่อนไข
  • จะหยุดทำงานและส่งคืนผลลัพธ์ทันทีที่เจอข้อมูลตัวแรกที่ผ่านเงื่อนไข (ช่วยประหยัดทรัพยากร)

7. Method every() (ตรวจสอบว่าทุกตัวตรงเงื่อนไขหรือไม่)

Method every() จะทำงานตรงกันข้ามกับ some() โดยจะตรวจสอบว่า ข้อมูลทุก ๆ ตัวใน Array ผ่านเงื่อนไขทั้งหมดหรือไม่:

const numbers = [1, 2, 3, 4, 5]
const allPositive = numbers.every((number) => number > 0) // true
const allLarge = numbers.every((number) => number > 3) // false

กลไกการทำงานของ every():

  • จะส่งคืนค่า true ก็ต่อเมื่อข้อมูลทุกตัวใน Array ผ่านเงื่อนไขทั้งหมด
  • จะส่งคืนค่า false ทันทีหากพบข้อมูลแม้เพียงตัวเดียวที่ไม่ผ่านเงื่อนไข
  • จะหยุดทำงานและส่งคืนผลลัพธ์ทันทีที่เจอข้อมูลตัวแรกที่ไม่ผ่านเงื่อนไข

8. Method reduce() (การประมวลผลสรุปยอดรวม)

Method reduce() เป็นเครื่องมือที่ทรงพลังและยืดหยุ่นที่สุด (แต่ก็มักสร้างความสับสนให้กับผู้เริ่มต้นมากที่สุดเช่นกัน) ทำหน้าที่สำหรับ ประมวลผลข้อมูลใน Array ทั้งหมดเพื่อสรุปยอดให้ออกมาเหลือเพียงค่าเดียว (เช่น การหาผลรวม หรือการนับจำนวน):

const numbers = [1, 2, 3, 4, 5]
const sum = numbers.reduce((accumulator, number) => {
  return accumulator + number
}, 0) // 0 คือค่าเริ่มต้น (Initial Value)
console.log(sum) // 15

กลไกการทำงานของ reduce():

  1. ค่าเริ่มต้น (Initial Value): คือค่าที่เรากำหนดไว้ใน Argument ตัวที่สอง (ในตัวอย่างคือ 0)
  2. ตัวสะสม (Accumulator): ตัวแปรที่เก็บผลรวมสะสม โดยในรอบแรกจะเริ่มจากค่าเริ่มต้น และในรอบถัด ๆ ไปจะเก็บค่าที่ส่งคืนมาจาก Function คอลแบ็ก
  3. ค่าปัจจุบัน (Current Value): ข้อมูลสมาชิกแต่ละตัวใน Array ที่กำลังถูกประมวลผลตามลำดับ
  4. ค่าที่ส่งคืน (Return Value): ผลลัพธ์ที่ได้จากการคำนวณในรอบนั้น ๆ ซึ่งจะถูกส่งไปเป็นค่า Accumulator ของรอบถัดไป

ตัวอย่างการทำงานแบบทีละขั้นตอน (Trace Log):

const sum = numbers.reduce((accumulator, number) => {
  console.log(`accumulator: ${accumulator}, number: ${number}`)
  return accumulator + number
}, 0)
// 0+1 → 1, 1+2 → 3, 3+3 → 6, 6+4 → 10, 10+5 → 15
console.log(sum) // 15
