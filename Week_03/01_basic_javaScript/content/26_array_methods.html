เมธอด Array

Array มีความสามารถสูงมากใน JavaScript และจะมีประโยชน์ยิ่งขึ้นเมื่อรู้จักเมธอดที่มีมาให้สำหรับทำงานกับ Array อย่างมีประสิทธิภาพ

เมธอด Array คืออะไร?

คือฟังก์ชันที่มีอยู่ใน Array ของ JavaScript เรียกใช้ด้วย dot notation เช่นเดียวกับการเข้าถึงพร็อพเพอร์ตีของ Object

const numbers = [1, 2, 3, 4, 5]
numbers.forEach((n) => {
  // ทำบางอย่างกับตัวเลขแต่ละตัว
})

เมธอด forEach

forEach น่าจะเป็นเมธอด Array ที่ใช้บ่อยที่สุด โดยวนผ่านสมาชิกแต่ละตัวและรันฟังก์ชันให้แต่ละตัว

const numbers = [1, 2, 3, 4, 5]
numbers.forEach((number) => console.log(number))
// แสดง 1, 2, 3, 4, 5

forEach พร้อม Index

forEach สามารถให้ Index ของสมาชิกแต่ละตัวได้:

const names = ["Kyle", "Sarah", "John"]
names.forEach((name, index) => {
  console.log(`${name} ${index}`)
})
// Kyle 0, Sarah 1, John 2

Index ตรงกับ bracket notation ที่ใช้เข้าถึงสมาชิก:

console.log(numbers[0]) // 1
console.log(numbers[1]) // 2

เมธอด map

map คล้าย forEach แต่มีความแตกต่างสำคัญคือ ส่ง Array ใหม่กลับมา จากค่าที่ฟังก์ชันส่งกลับ:

const numbers = [1, 2, 3, 4, 5]
const doubled = numbers.map((number) => {
  return number * 2
})
console.log(doubled) // [2, 4, 6, 8, 10]
console.log(numbers) // ต้นฉบับไม่เปลี่ยน

ประเด็นสำคัญของ map:

  • ส่ง Array ใหม่กลับ - ไม่แก้ไขต้นฉบับ
  • ความยาวเท่าเดิม - Array ใหม่มีจำนวนสมาชิกเท่าเดิม
  • แปลงสมาชิกแต่ละตัว - สมาชิกแต่ละตัวกลายเป็นสิ่งใหม่ได้

กรณีใช้งานทั่วไป

const people = [
  { id: 1, name: "Alice" },
  { id: 2, name: "Bob" },
  { id: 3, name: "Charlie" },
]
const names = people.map((person) => person.name)
console.log(names) // ["Alice", "Bob", "Charlie"]

เมธอด filter

filter สร้าง Array ใหม่ที่มีเฉพาะสมาชิกซึ่งผ่านการทดสอบ ส่ง true เพื่อเก็บ และ false เพื่อตัดออก

const numbers = [1, 2, 3, 4, 5]
const smallNumbers = numbers.filter((number) => number <= 2)
console.log(smallNumbers) // [1, 2]
console.log(numbers) // ต้นฉบับไม่เปลี่ยน

เมธอด find

find คล้าย filter แต่ส่งกลับเฉพาะสมาชิก ตัวแรก ที่ตรงเงื่อนไข แล้วหยุดค้นหา

const numbers = [1, 2, 3, 4, 5]
const firstBigNumber = numbers.find((number) => number > 2)
console.log(firstBigNumber) // 3

เมธอด some

some ตรวจว่า อย่างน้อยหนึ่ง สมาชิกผ่านการทดสอบหรือไม่ และส่ง true หรือ false กลับ

const numbers = [1, 2, 3, 4, 5]
const hasLargeNumber = numbers.some((number) => number > 3) // true
const hasNegativeNumber = numbers.some((number) => number < 0) // false

some ทำงานอย่างไร:

  • ส่ง true หากมีสมาชิกตัวใดผ่าน
  • ส่ง false หากไม่มีสมาชิกผ่าน
  • หยุดทันทีเมื่อพบตัวที่ผ่าน

เมธอด every

every ตรงข้ามกับ some โดยตรวจว่าสมาชิก ทั้งหมด ผ่านการทดสอบหรือไม่

const numbers = [1, 2, 3, 4, 5]
const allPositive = numbers.every((number) => number > 0) // true
const allLarge = numbers.every((number) => number > 3) // false

every ทำงานอย่างไร:

  • ส่ง true หากทุกตัวผ่าน
  • ส่ง false หากมีตัวใดไม่ผ่าน
  • หยุดทันทีเมื่อพบตัวที่ไม่ผ่าน

เมธอด reduce

reduce เป็นเมธอดที่ทรงพลังที่สุด (และชวนสับสนที่สุด) โดย “ลด” Array ให้เหลือค่าเดียวด้วยฟังก์ชันที่รวบรวมผลลัพธ์

const numbers = [1, 2, 3, 4, 5]
const sum = numbers.reduce((accumulator, number) => {
  return accumulator + number
}, 0) // 0 คือค่าเริ่มต้น
console.log(sum) // 15

reduce ทำงานอย่างไร:

  1. ค่าเริ่มต้น: อาร์กิวเมนต์ตัวที่สอง
  2. Accumulator: เริ่มจากค่าเริ่มต้น แล้วกลายเป็นค่าที่ส่งกลับ
  3. สมาชิกปัจจุบัน: สมาชิกแต่ละตัวตามลำดับ
  4. ค่าที่ส่งกลับ: กลายเป็น Accumulator ใหม่ในรอบถัดไป

ติดตามตัวอย่างผลรวม:

const sum = numbers.reduce((accumulator, number) => {
  console.log(`accumulator: ${accumulator}, number: ${number}`)
  return accumulator + number
}, 0)
// 0+1 → 1, 1+2 → 3, 3+3 → 6, 6+4 → 10, 10+5 → 15
console.log(sum) // 15
