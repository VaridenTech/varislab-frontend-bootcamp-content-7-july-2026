ชนิดข้อมูลพื้นฐานและการดำเนินการ (Primitive Types & Operations)

JavaScript มี ชนิดข้อมูลพื้นฐาน (Primitive Types) อยู่หลายชนิด ซึ่งเปรียบเสมือนอิฐบล็อกพื้นฐานในการประกอบกันเป็นข้อมูลทั้งหมดในโปรแกรมของคุณ

ชนิดข้อมูลพื้นฐานสำคัญ 3 ชนิด

1. ตัวเลข (Number)

ใช้สำหรับแทนค่าตัวเลขทุกรูปแบบ ไม่ว่าจะเป็นจำนวนเต็ม (Integer) ทศนิยม (Float) จำนวนติดลบ หรือแม้กระทั่งค่าตัวเลขพิเศษ (เช่น Infinity)

-10
25
19.99
0
Infinity

2. ข้อความ (String)

ใช้สำหรับแทนข้อมูลที่เป็นตัวอักษรหรือข้อความ โดยจะต้องเขียนอยู่ภายใต้เครื่องหมายคำพูด (Quotes)

'Sarah'
"Hello, world!"
''
""
  • สามารถสร้างได้ทั้งเครื่องหมายคำพูดเดี่ยว (Single Quotes) หรือเครื่องหมายคำพูดคู่ (Double Quotes)
  • สามารถเป็นข้อความว่าง (Empty String) ได้ (มีเพียงเครื่องหมายคำพูดแต่ไม่มีตัวอักษรข้างใน)

3. Boolean (Boolean)

ใช้สำหรับแทนค่าทางตรรกศาสตร์ที่มีเพียง 2 ค่าเท่านั้น คือ จริง (true) หรือ เท็จ (false)

true
false

การดำเนินการพื้นฐาน (Operations)

การดำเนินการ (Operations) คือสิ่งที่เราสามารถทำกับข้อมูลพื้นฐานเหล่านั้นได้ โดยข้อมูลแต่ละชนิดจะรองรับตัวดำเนินการ (Operators) ที่แตกต่างกันออกไป

การดำเนินการกับ Number (Arithmetic & Comparison)

  • การคำนวณทางคณิตศาสตร์ (Arithmetic Operations)

    // การคำนวณพื้นฐาน
10 + 3 // 13
10 - 3 // 7
10 * 3 // 30
10 / 3 // 3.333...

// ลำดับการคำนวณทางคณิตศาสตร์ (PEMDAS)
2 + 3 * 4 // 14 (ไม่ใช่ 20)
(2 + 3) * 4 // 20

  • การเปรียบเทียบ (Comparison Operations)

    85 == 92 // false
85 != 92 // true

85 < 92 // true
85 > 92 // false
85 <= 85 // true
92 >= 90 // true

การดำเนินการกับ String (Concatenation & Comparison)

  • การต่อข้อความ (String Concatenation)

    "Kyle" + " " + "Cook" // "Kyle Cook"

  • การเปรียบเทียบข้อความ (String Comparison)

    "apple" == "apple" // true
"apple" != "apple" // false

// เปรียบเทียบตามลำดับตัวอักษร (Alphabetical Order)
"apple" < "banana" // true ("apple" มาก่อน "banana" ตามพจนานุกรม)

การดำเนินการกับ Boolean (Logical Operations)

  • ตัวดำเนินการทางตรรกศาสตร์ (Logical Operators)

    true && false // false (AND)
true || false // true (OR)
!true // false (NOT)

  • การเปรียบเทียบค่า Boolean

    true == true // true
true == false // false
true != false // true

การตรวจสอบชนิดข้อมูลด้วย typeof

เราสามารถตรวจสอบชนิดข้อมูลของค่าหรือตัวแปรใด ๆ ได้โดยใช้ตัวดำเนินการ typeof:

typeof 42 // "number"
typeof "Hello" // "string"
typeof true // "boolean"

typeof 1 == typeof 2 // true
