# CSS Box Model คืออะไร

ก่อนจะตกแต่งหน้าเว็บ วันนี้เราต้องเข้าใจ concept สำคัญของ CSS ก่อน นั่นคือ **Box Model**

ใน CSS ทุก element บนหน้าเว็บจะถูกมองเป็น “กล่อง” หนึ่งกล่อง

กล่องหนึ่งกล่องมีส่วนประกอบหลัก ๆ คือ:

| ส่วนประกอบ  | ความหมาย                                             |
| ----------- | ---------------------------------------------------- |
| **Content** | เนื้อหาข้างใน เช่น ข้อความ รูปภาพ หรือปุ่ม           |
| **Padding** | ระยะห่างด้านในกล่อง ระหว่าง content กับ border       |
| **Border**  | เส้นขอบของกล่อง                                      |
| **Margin**  | ระยะห่างด้านนอกกล่อง ระหว่างกล่องนี้กับ element อื่น |

พูดง่าย ๆ คือ:

```text
Margin
└── Border
    └── Padding
        └── Content
```

ถ้าเราไม่เข้าใจ **Box Model** เวลา layout เพี้ยน เราจะไม่รู้ว่าเพี้ยนเพราะอะไร เช่น:

* `padding` ดันเนื้อหาข้างใน
* `margin` ดัน element ออกจากกัน
* `border` ทำให้กล่องมีขอบเพิ่ม
* `width` หรือ `height` ทำให้ขนาดกล่องเปลี่ยน

ดังนั้นก่อนจะเขียน CSS layout ให้ดี เราต้องเข้าใจก่อนว่า element ทุกตัวบนหน้าเว็บคือกล่อง และกล่องนั้นประกอบด้วย **content, padding, border และ margin**
