# W03.01 — บทนำสู่ JavaScript: สเปกบทเรียน

## เป้าหมาย

สร้างเนื้อหาภาษาไทยสำหรับผู้สอนใช้สอนนักเรียนในคาบ JavaScript ครั้งแรก โดยอ้างอิงหัวข้อ “What Is JavaScript?” จาก Web Dev Simplified และปรับระดับภาษาให้เหมาะกับผู้เริ่มต้นในคอร์สนี้

## ไฟล์และลำดับ

สร้างโฟลเดอร์ `Week_03/01_basic_javaScript/content` และเริ่มด้วยไฟล์ `01_what_is_javascript.md` เพื่อให้เนื้อหาถูกสอนตามลำดับหมายเลขเมื่อเพิ่มบทเรียนต่อไป

## รูปแบบบทเรียน

เอกสารเป็น teaching script สำหรับเวลา 45–60 นาที ประกอบด้วย:

1. ชื่อบทเรียนและเป้าหมายการเรียนรู้
2. บทเปิดคาบที่เชื่อม HTML, CSS และ JavaScript
3. คำอธิบาย JavaScript แบบง่าย พร้อมตัวอย่างสิ่งที่ผู้เรียนพบในเว็บ
4. ขอบเขตการใช้งาน: browser, server, desktop และ mobile
5. คำอธิบาย “scripting language” ระดับเริ่มต้น โดยไม่ทำให้เข้าใจผิดว่า JavaScript ไม่มีขั้นตอน compile เลยในทุก runtime
6. จุดแข็งในการทดลองและตรวจข้อผิดพลาดผ่าน browser
7. กิจกรรมสั้น 5 นาที และคำถามเช็กความเข้าใจ
8. สรุปท้ายคาบและสะพานไปยังบทถัดไป

## แนวทางภาษาและเนื้อหา

- ใช้ภาษาไทยเป็นหลัก คงศัพท์เทคนิคและชื่อเทคโนโลยีที่จำเป็นเป็นภาษาอังกฤษใน backticks
- มุ่งให้ผู้เรียนเข้าใจภาพรวม ไม่บังคับให้จำชื่อ framework หรือ runtime ทั้งหมด
- ระบุว่า JavaScript กับ Java เป็นคนละภาษา และอธิบายคำว่า “interpreted” อย่างระมัดระวัง: code ของผู้เรียนถูกรันผ่าน JavaScript engine และ engine สมัยใหม่อาจมีการ optimize ภายใน
- รักษาลิขสิทธิ์โดยเรียบเรียงและเสริมบริบทการสอนเอง ไม่คัดลอกบทความต้นฉบับเป็นข้อความยาว

## การตรวจสอบ

ตรวจว่าโครงสร้างไฟล์ถูกต้อง, ไฟล์ใช้ UTF-8, หัวข้อเรียงตามลำดับการสอน, และตัวอย่าง code สามารถอ่านและใช้ใน browser console ได้
