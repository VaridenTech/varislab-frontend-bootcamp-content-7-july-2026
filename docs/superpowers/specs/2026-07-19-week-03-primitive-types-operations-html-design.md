# W03.02 Primitive Types and Operations — Thai HTML Translation

## Objective

Create a faithful Thai translation of Web Dev Simplified's “Primitive Types and Operations” lesson in HTML format for classroom use.

## Deliverable

Create exactly one lesson file:

`Week_03/01_basic_javaScript/content/02_primitive_types_and_operations.html`

## Content and Structure

- Use a complete UTF-8 HTML document with `lang="th"`.
- Preserve the source lesson's heading hierarchy, paragraph order, lists, and code-example order.
- Translate the title, headings, prose, list items, code comments, and explanatory comments into natural Thai.
- Preserve JavaScript syntax, keywords, operators, literal values, type names, example names, and results.
- Include the source sections about `number`, `string`, `boolean`, operations for each type, and `typeof`.
- Do not introduce concepts absent from the source, including strict equality, type coercion, variables, or additional primitive types.

## Presentation

Use plain semantic HTML only. Do not add CSS, JavaScript interactions, navigation, activities, exercises, instructor notes, schedules, summaries, or other teaching material that does not appear in the source.

## Attribution

Include a concise footer identifying the document as a Thai translation, linking to the original Web Dev Simplified lesson, and identifying the source content's CC BY-NC 4.0 license and the code samples' Apache 2.0 license. This is the only content added beyond the translated lesson and is required by the source license.

## Verification

- Confirm the document parses as HTML and declares Thai and UTF-8.
- Compare every source heading, paragraph, list item, and code example against the translation.
- Confirm that no source concept or example is omitted and no extra instructional content is added.
- Confirm the required attribution and license links are present.
